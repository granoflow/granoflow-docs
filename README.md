# Granoflow Public Documentation

This repository stores public Granoflow documentation source that can be read directly on GitHub.

- Product manual source: [manual/README.md](manual/README.md)
- Published manual site: [https://granoflow.com/manual/](https://granoflow.com/manual/)
- Legal documents: [docs/](docs/)

The public manual content is limited to documentation text and public screenshots. Website implementation files, Astro/Starlight configuration, custom CSS, client scripts, build scripts, deployment settings, private notes, and secrets belong outside this repository.
