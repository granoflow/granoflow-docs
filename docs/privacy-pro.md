### 2.2 User Content (Locally Encrypted Before Synchronization)

With your explicit consent, GranoFlow Pro may synchronize certain application data to our servers, including but not limited to:

* Fields and values

## 4. Data Storage and Security

We employ reasonable technical and organizational measures to protect your data, including but not limited to:

While we take reasonable security measures, no network or storage system can guarantee absolute security.
