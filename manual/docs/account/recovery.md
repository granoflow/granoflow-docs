账号恢复解决一个很具体的问题：**你在 App Store 或 Google Play 购买了 GranoFlow 会员权益，但不确定这笔购买连接的是哪个账号**。

它不是账号删除、退出登录、订阅恢复购买或数据恢复的入口。

![账号恢复界面](../../screenshots/zh-cn/account-recovery-main.png)

## 什么时候用账号恢复

✅ 适合的情况：

- 你确定在商店买过 GranoFlow，但当前账号看不到会员权益
- 你不确定当时用的是哪个邮箱注册的

❌ 不适合的情况：

- 你只是想恢复当前平台的订阅 → 用"恢复购买"
- 你的本地数据丢失了 → 用"备份与恢复"
- 你想找回加密密钥 → 用"加密与恢复密钥"

## 操作步骤

1. 在登录页面或账号相关页面找到"账号恢复"链接
2. 填写你希望连接的邮箱
3. 确认这台设备可以访问当时购买所在的商店账号
4. 提交申请，等待系统核对

## 可能的结果

| 结果 | 含义 |
|------|------|
| 申请已提交 | 系统收到申请，按提示等待或查看邮件 |
| 没有历史记录 | 当前平台下找不到可用的购买记录 |
| 记录不匹配 | 商店记录和邮箱无法通过校验连接 |
| 暂时失败 | 网络或服务暂时不可用，可稍后重试 |

:::caution[别急着删数据]
如果你怀疑自己登录了错误账号，**先不要删除本机数据**。确认当前登录邮箱，再查订阅页和设备管理。
:::
