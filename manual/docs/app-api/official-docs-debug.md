官方文档调试用于让 `granoflow.com` 的文档页面在限定时间内连接当前设备。它适合在阅读官方文档或 Swagger 页面时，临时验证本机接口。

桌面端更适合使用官方文档调试。手机端如果要从电脑浏览器调试手机上的 Granoflow，应使用手机显示的局域网地址，并确认电脑和手机在同一 Wi-Fi。手机端限制见 [手机端局域网连接](/manual/app-api/mobile-lan-connection/)。

## 开启调试

1. 在 App 中进入“本机接口服务”。
2. 先开启“开启接口服务”。
3. 在“官方文档调试”区域点击“开启调试”。
4. 复制临时访问码。
5. 回到官方文档页面，按页面提示填入访问码或完成调试。

桌面端调试开启后会显示剩余时间。关闭或到期后，官方文档页面不能继续使用这次临时访问码连接当前设备。手机端不要把这个倒计时理解成后台持续运行承诺。

## 什么时候使用

当你从官方文档页面发起调试请求时，使用官方文档调试。

当你只是让本机脚本或 AI 工具连接当前设备时，优先使用“本机地址”和常规访问码设置，不一定需要开启官方文档调试。

## 安全边界

临时访问码只用于官方文档调试场景。不要把它长期保存到第三方工具里，也不要把它发到公开聊天、论坛或截图中。

如果你已经完成调试，点击“关闭调试”。如果忘记关闭，到期后也会自动失效。
