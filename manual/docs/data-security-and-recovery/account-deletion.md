---
title: "账号删除与数据处理"
description: "删除账号会清除云端数据，但不会自动清除每台设备上的本地副本——操作前先搞清楚影响范围。"
---

删除 GranoFlow 账号是一个不可逆操作。在操作前，有几件事需要搞清楚。

## 删账号会发生什么

| | 删账号后 |
| --- | --- |
| 云端数据 | 在宽限期后永久删除 |
| 各设备本地数据 | **不会自动删除** |
| 订阅权益 | 账号删除后权益失效 |
| 可以撤销吗 | 宽限期内可以取消，过后不可逆 |

## 删账号前的检查清单

- [ ] 确认你不再需要云端同步或多设备共享
- [ ] 如果你还想保留数据，先导出本地备份
- [ ] 如果你有正在进行中的订阅，了解退款政策
- [ ] 如果你有多台设备，了解每台设备上的数据状态

## 删账号 vs 退出登录

- **退出登录**：断开这台设备和账号的连接，本地数据保留，可以重新登录
- **删除账号**：注销账号，云端数据在宽限期后删除，不可恢复

如果你只是暂时不想用 GranoFlow，退出登录就够了。删账号是永久操作。

## 宽限期

账号删除申请提交后，通常有一段宽限期（具体时长见 App 内说明）。宽限期内你可以取消删除申请，让账号恢复正常。宽限期结束后，删除操作不可撤销。

:::caution[本地数据不会自动消失]
删除账号后，你每台设备上的本地 GranoFlow 数据仍然存在，直到你手动卸载 App 或清除 App 数据。如果你不想要这些本地数据，需要手动清除。
:::
