---

GranoFlow 的云端同步数据是端对端加密的。加密密钥就像保险箱的钥匙——**没有它，连 GranoFlow 自己的服务器也无法读取你的数据**。

但这把钥匙也意味着：**如果你自己丢了，就真的找不回来了**。

![加密与恢复密钥界面](../../screenshots/zh-cn/data-encryption-recovery-key.png)

## 密钥 vs 登录密码，有什么区别

| | 登录密码（或验证邮件） | 加密/同步密钥 |
| --- | --- | --- |
| 用来做什么 | 证明你是谁 | 打开云端加密数据 |
| 忘了怎么办 | 重新发验证邮件 | **无法找回** |
| 换了会怎样 | 只影响登录 | 影响是否能访问云端数据 |

## 我的密钥在哪里

在 GranoFlow 设置 → 数据/安全/同步 里可以查看和保存当前设备的云端同步密钥。

**强烈建议：把密钥抄下来，或保存到你的密码管理器里。**

## 在新设备上需要密钥的情况

当你：

- 换了手机或电脑
- 重装了 GranoFlow
- 出现了"同步密钥不匹配"的提示

GranoFlow 会要求你输入旧设备上的云端同步密钥，才能访问云端的已有数据。

## 输入密钥后会发生什么

GranoFlow 先验证密钥能不能打开当前云端数据：

- **密钥匹配，云端和本地是同一份数据** → 直接连接同步
- **密钥匹配，但本地有新数据** → 显示选择界面，让你决定保留哪份数据
- **密钥不对** → 不会改变任何数据，让你重新输入

## 忘了密钥怎么办

先不要慌，按顺序检查：

1. **旧设备还能用吗？** → 在旧设备上打开 GranoFlow，找到密钥并复制
2. **密码管理器里有吗？** → 检查你的密码管理器
3. **旧设备有但打不开 App？** → 联系 GranoFlow 支持

如果以上都没有，云端加密数据可能无法恢复。本地备份（如果有的话）仍然可用。

## "用云端密钥替换本地密钥"是什么意思

这个选项用于：当前设备有一份本地数据，但你决定让这台设备切换到使用云端的加密方案。

**操作前务必确认**：你有当前云端数据对应的完整密钥，以及你清楚本机和云端哪份数据更重要。

:::caution[密钥不是密码，不能重置]
加密密钥丢失后，GranoFlow 无法帮你重置或找回。现在就去保存你的密钥，不要等到用到的时候才后悔。
:::
