同步把你在一台设备上的变化，带到其他登录了同一账号的设备上。

![同步状态界面](../../screenshots/zh-cn/data-sync-status-main.png)

## 同步什么、不同步什么

✅ 同步的内容：

- 任务（标题、截止日期、标签、状态……）
- 项目和里程碑
- 回顾记录
- 图片和附件（在网络允许的情况下）

⚠️ 同步不是备份：

- **你删了，其他设备也会删** — 同步是双向的
- **没有版本历史** — 同步不记录"3天前的状态"
- **图片可能延迟** — 文字先同步，图片可能晚一步

## 同步的常见状态

| 状态 | 含义 |
| --- | --- |
| 同步中 | 正在上传或下载变更 |
| 已同步 | 当前设备数据和云端一致 |
| 等待中 | 有变更在排队，通常是网络问题 |
| 错误 | 同步遇到了问题，需要检查账号或密钥 |

## 离线和服务不可达时会怎样

GranoFlow 的本地数据会继续可用。你仍然可以记录任务、整理项目、写回顾、搜索内容，并导出本地备份。

如果你开启了「离线模式」，同步、登录、订阅确认和恢复购买会暂时不可用。你点击这些在线功能时，App 会提示当前无法连接对应服务；关闭离线模式后可以再重试。

如果设备有网络，但同步服务暂时不可达，App 不会阻断本地使用。已经在本机的数据仍然可以编辑；需要上传或下载到云端的变化会等你稍后再次同步。

## 新设备加入同步

如果你安装了新手机或重装了 App，需要用旧设备的**云端同步密钥**才能加入已有的云端数据。

详细步骤见 → [新设备同步已有云端数据](/manual/data-security-and-recovery/new-device-sync/)

:::caution[同步不替代备份]
定期导出本地备份。误删的任务不能靠同步恢复——云端早就跟着删了。
:::
