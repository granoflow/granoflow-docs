title: "桌面端安装"

---

如果你想在电脑上使用 GranoFlow，请先从 GranoFlow 官网下载安装包，安装后用同一个账号登录。桌面端可以留在电脑上使用，也可以通过快捷键呼出，不需要每次都先打开浏览器。

## 下载安装

打开 [GranoFlow 官网](https://granoflow.com)，下载适合你电脑系统的安装包：

- **macOS**：下载 `.dmg` 文件，打开后把 GranoFlow 拖到「应用程序」文件夹。
- **Windows**：下载 `.exe` 安装包，然后按照安装向导一步步完成安装。

## macOS 提示「无法打开」怎么办

第一次打开 GranoFlow 时，macOS 可能会提示它来自未知开发者，导致你不能直接打开。

可以这样处理：

1. 打开 Finder，找到 GranoFlow 应用。
2. 右键点击 GranoFlow。
3. 选择「打开」。
4. 系统再次弹出确认窗口时，再点击「打开」。

也可以从系统设置里处理：打开「系统设置」→「隐私与安全性」，找到 GranoFlow 的拦截记录，然后点击「仍然打开」。

## Windows 提示 SmartScreen 怎么办

第一次运行 GranoFlow 时，Windows Defender SmartScreen 可能会弹出警告。

在这个提示里，先点击「更多信息」，再点击「仍要运行」。

## 安装完成后

安装完成后，打开 GranoFlow，并用你的账号登录。如果这个账号之前已经有云端数据，GranoFlow 会开始同步这些数据。

:::note[桌面端和手机端共用同一账号]
在桌面端和手机端登录同一个账号后，数据会通过云端同步，在不同设备之间共享。
:::

如果你需要在终端里调用 GranoFlow，下一步阅读 [命令行工具](/manual/desktop/command-line-tool/)。
