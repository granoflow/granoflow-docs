---
title: "桌面端安装"
description: "下载、安装 GranoFlow 桌面端，以及首次打开时遇到系统拦截怎么处理。"
---

桌面端让 GranoFlow 常驻在你的电脑上，随时用快捷键呼出，不需要每次打开浏览器。

## 下载安装

从 [GranoFlow 官网](https://granoflow.com) 下载对应系统的安装包：

- **macOS**：下载 `.dmg` 文件，打开后拖入应用程序文件夹
- **Windows**：下载 `.exe` 安装包，按照安装向导完成

## macOS 提示"无法打开"怎么处理

第一次打开时，macOS 可能提示"来自未知开发者"。

解决方法：

1. 在 Finder 里右键点击应用 → 选择"打开"
2. 再次点击"打开"确认

或者：系统设置 → 隐私与安全性 → 找到 GranoFlow 的拦截记录 → 点击"仍然打开"。

## Windows 提示 SmartScreen 怎么处理

Windows Defender SmartScreen 可能弹出警告。点击"更多信息"→"仍要运行"即可。

## 安装完成后

首次打开后用你的账号登录，GranoFlow 会自动开始同步云端数据（如果你之前有云端数据的话）。

:::note[桌面端和手机端共用同一账号]
登录同一账号，数据会通过云端同步在桌面端和手机端之间共享。
:::
