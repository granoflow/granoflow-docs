---
title: "Offline use and updates"
description: "GranoFlow desktop works without internet — understand the boundaries of offline mode and how updates affect your data."
---

When you have no internet connection, you can still use GranoFlow desktop to view and edit content already available on this computer. After the connection comes back, changes made offline are synced to the cloud. Version updates usually update the app itself and do not delete your local data or account settings.

## What you can do offline

Without a network connection, the GranoFlow desktop app can still do these things:

- View, add, and edit tasks
- Write journal entries and reviews
- View projects and milestones

These actions are saved on this computer first. In other words, you can keep organizing tasks and writing things down without waiting for the network to recover.

You cannot do these things while offline:

- Sync new changes from this computer to other devices
- Sync new changes from the cloud or other devices to this computer

So if you edit a task while this computer is offline, your phone or another computer will not see the edit immediately. Likewise, new content from other devices will not appear on this computer immediately.

## After reconnecting

After you reconnect, GranoFlow automatically syncs the changes you made offline to the cloud. You usually do not need to export or re-enter that content manually.

If multiple devices changed content while offline, syncing may require changes to be merged. If that happens, follow the result or prompt shown in the app.

## Version updates

GranoFlow desktop will notify you when a new version is available.

- **If you skip the update**: you can keep using the current version, but new features may not be available; if the server later makes an incompatible change, you may need to update before continuing
- **Do updates delete my data?**: No. Updates only replace the app files. Your local data and account settings are not affected

:::note[Forced updates]
If the server makes an incompatible change, GranoFlow may require an update before you can continue.
:::
