## Tasks and features

**What does GranoFlow do?**  
Task management, journaling, reviews, and project tracking — with a focus on connecting what you *want to do* to what you *actually do*, through domains, values, projects, milestones, and reviews.

**How is it different from a regular to-do app?**  
GranoFlow cares about *why* tasks matter, not just checking them off. You can link tasks to long-term projects and values, and use reviews to see what's actually moving forward.

**Is there a task limit?**  
No. Local tasks have no limit.

## AI

**Will AI automatically edit my tasks?**  
No. All AI suggestions require your explicit confirmation before anything is written. Unaccepted suggestions have zero effect on your task data.

**Does AI read all my data?**  
No. Data only enters the AI pipeline when you actively trigger an AI feature. Browsing tasks, writing journal entries, and doing reviews involve no AI.

**Do AI features require a subscription?**  
Most AI features (title parsing, clipboard assistant, redaction terms) are subscriber-only.

## Data and sync

**Where is my data stored?**  
Primarily on your device. If you enable cloud sync, it's also stored in the cloud.

**Is sync the same as backup?**  
No. Sync mirrors your actions — if you delete something, the cloud deletes it too. Backup captures a point-in-time snapshot and can restore accidentally deleted data. Back up regularly.

**If I switch phones, will my data survive?**  
You need cloud sync (subscriber feature) to move data to a new device. Sign in with the same account on the new device and enter your cloud sync key from the old device.

**I lost my encryption key — what now?**  
Check your old device or password manager first. If it's truly gone, your encrypted cloud data may be unrecoverable. **Save your key now** — find it under Settings → Data/Security.

## Subscription

**Will my data disappear if my subscription lapses?**  
No. Local data stays intact. Some subscriber features will pause, but your data is not affected.

**I bought on iOS — can I use it on Android?**  
Not automatically. iOS and Android purchases are independent platform transactions and cannot be transferred between them.

**How do I restore a purchase?**  
Go to Settings → Subscription → Restore purchases. Make sure the account you're signed into matches the one you used to purchase.

## Desktop

**Does closing the window quit the app?**  
No. Closing the window hides it; GranoFlow keeps running in the background. To fully quit: right-click the Dock icon on macOS → Quit, or right-click the tray icon on Windows → Quit.

**Is desktop data the same as mobile data?**  
Yes, if you're using the same account with cloud sync enabled (subscriber feature).
