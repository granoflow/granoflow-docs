Device preferences adjust feedback, protection, and reminders on this device. They usually do not change your tasks, projects, tags, or review records, and they should not be understood as settings that automatically sync to every device.

If you switch to a new phone or computer, some display and reminder habits may need to be confirmed again on the new device.

## Timer

Timer-related settings control local feedback while recording focus time, such as whether to play a sound or how state changes are indicated.

These settings only affect the timer experience on the current device. Focus time that has already been recorded is business data, which is different from local preferences such as timer sound.

## App Lock

App lock adds access protection on the current device.

It is useful on shared devices, when stepping away temporarily, or when using GranoFlow on mobile. App lock is not your [account](/manual/en/account/overview/) password, and it does not replace a recovery key or system-level security settings.

## Task Reminder Banners

Task reminder banners control whether lightweight task reminder prompts are shown on the current device.

Turning banners off does not delete tasks or clear reminder times inside tasks. If you run into cross-device reminder problems, check the current device's notification permissions together with [multi-device sync](/manual/en/data-security-and-recovery/sync/) status.

## Inbox Move Hints

Inbox Move Hints give success feedback when a task leaves Inbox for Today, a project, a milestone, or another destination that is not immediately visible on the current page.

Completion, postponing, dragging, and other task actions that are already reflected on the page do not show success hints. Turning these hints off does not cancel the action itself, and error messages still appear.
