If you want GranoFlow to feel easier to read and use, go to Settings -> Appearance to change the language, theme, and font size. These settings only affect what you see on the current device. They do not change your tasks, projects, tags, review records, or synced data.

## Change Language

Open Settings, go to Appearance, then choose Language. Select the interface language you want to use.

<!-- manual-screenshot:id=interface-settings-overview-main -->
![Language, theme, and font interface screenshot](../../../screenshots/en/interface-settings-overview-main.png)

Language changes only affect the app's built-in interface text. They do not automatically translate content you wrote yourself, such as task titles, project names, tags, notes, or review content.

For example, if you switch the interface from Chinese to English, Chinese tasks you already wrote will stay in Chinese.

If you use GranoFlow on multiple devices, each device can use the language that fits that device. Language preference should not be understood as account-level data, and it should not affect business records in [multi-device sync](/manual/en/data-security-and-recovery/sync/).

## Change Theme

Open Settings, go to Appearance, then choose Theme. You can choose light, dark, or system.

Theme only affects how GranoFlow looks on the current device. You can use dark theme on desktop and light theme on mobile; this does not change any tasks, projects, review records, or [account](/manual/en/account/overview/) status.

If a platform is set to follow the system, the actual appearance depends on the current device's system appearance setting.

## Adjust Font Size

Open Settings, go to Appearance, then choose Font size. Use it to make interface text larger or smaller.

Font size is useful to set separately per device. For example, you may use larger text on mobile for easier reading and keep a more compact display on desktop. It only changes presentation. It does not change the data you wrote, and it does not affect [backup and restore](/manual/en/data-security-and-recovery/backup-and-restore/) contents.
