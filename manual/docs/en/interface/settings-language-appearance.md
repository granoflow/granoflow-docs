Language, theme, and font size determine how you read and operate GranoFlow on the current device. They are display experience settings. They do not change the meaning of tasks, projects, tags, review records, or synced data.

## Change Language

Go to Settings -> Appearance -> Language, then choose the interface language you want to use.

Changing language only changes the app interface copy. It does not automatically translate task titles, project names, tags, notes, or review content you wrote yourself. For example, if you switch the interface from Chinese to English, Chinese tasks you already wrote will remain Chinese.

If you use GranoFlow on multiple devices, each device can choose the language that fits its own environment. Language preference should not be understood as account-level data, and it should not affect business records in [multi-device sync](/manual/en/data-security-and-recovery/sync/).

## Change Theme

Go to Settings -> Appearance -> Theme, then choose light, dark, or system.

Theme only affects how the current device looks. You can use dark theme on desktop and light theme on mobile; this does not change tasks, projects, review records, or [account](/manual/en/account/overview/) status.

If a platform uses the system setting, the actual appearance depends on the current device's system appearance.

## Adjust Font Size

Go to Settings -> Appearance -> Font size to make interface text larger or smaller.

Font size is useful to set per device. For example, you can use larger text on mobile for easier reading and keep a more compact display on desktop. It only changes presentation, not the data you wrote, and it does not affect [backup and restore](/manual/en/data-security-and-recovery/backup-and-restore/) contents.
