---
title: "Quick Start: Run the First Practice in 5 Minutes"
description: "Do one minimal loop without waiting for a perfect state: capture, connect value, turn into a startable task, complete, and review."
---

If you want to start now, you do not need full theory first. Run one minimal loop:

> Capture one thing → Connect one value → Turn it into one task
> → Complete → Review

This loop is enough for your first practice.
You do not need a full system before acting.

## Step 1: Capture one real thing

Tap `+` and write the one thing taking your attention.

It can be small:

- Reply to one email
- Edit one line of copy
- Organize one file
- Book one health check

Capture first, classify later. If uncertain, leave it in inbox.

## Step 2: Ask which value it connects to

Ask one question:

> If I choose to do this, what direction that matters to me does it serve?

Examples:

- Reply to client email → reliable delivery
- Run 20 minutes → long-term care for my body
- Draft manual copy → sustained expression and creation

One honest sentence is enough.

## Step 3: Turn it into a startable task

Avoid:

> Make the manual good / Improve health / Learn English

Use:

- Draft the quick-start section
- Stretch for 5 minutes
- Listen to English for 10 minutes and write 3 lines

Rule: **if you can read it and start immediately, it is ready.**

## Step 4: Complete one small action

Pick the easiest task to start and mark it complete.

Completion is not a grade. It records a fact:

> This happened.

That fact becomes usable feedback in review.

## Step 5: Do a 1-minute review

At day end, answer:

```text
What did I complete today?
What direction/value did it serve?
What is the next step?
```

Review is not self-judgment. It turns action into continuation clues.

## 30-Second Flow Readiness Check

Before starting, check:

- Is the goal clear?
- Is challenge appropriate (not too big or too small)?
- Is feedback visible after completion?
- Can I return after interruption?

If 2 or more are "no", shrink the task further.
