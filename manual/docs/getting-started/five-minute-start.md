当你想用最短路径进入 GranoFlow 这款个人规划应用时，从这页开始。不需要先重建整个生活系统。

## 先捕捉一件事

打开快速新增，写下脑中最清楚的一个任务或想法。第一分钟可以把 GranoFlow 当成 Todo 应用：先记下来，再决定它属于哪里。

## 重要时再连接

如果这件事属于更长方向，把它连接到项目或里程碑。这就是一个日计划动作如何成为目标规划、项目规划器和个人时间结构的一部分。

## 回顾，不给自己打分

一天结束前，看一眼完成了什么，再写下下一步。日回顾不是分数、连续打卡或羞辱机制，而是让长期目标和复盘保持连接的安静方式。
