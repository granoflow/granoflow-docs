当前设备偏好里的设置，影响的是「这台设备上怎么用 GranoFlow」，而不是你的任务、项目或回顾数据。

换了新设备，这里的设置不会跟过去——需要在新设备上重新配置。

![当前设备偏好界面](../../screenshots/zh-cn/interface-device-preferences-main.png)

## 各项设置说明

**计时器**

控制专注计时时的本机反馈，比如提示音和状态显示。这些只影响用计时器的体验，不影响已经记录的投入时间数据。

**应用锁**

开启后，打开 GranoFlow 需要通过本机验证（Face ID / 指纹 / PIN）。

适合在共享设备或临时离开时保护内容不被偷看。注意：应用锁不是账号密码，也不能替代加密密钥或系统级安全设置。

**任务提醒横幅**

控制是否显示任务提醒的轻量提示横幅。

关闭横幅不会删除任务，也不会清除任务里设置的提醒时间。遇到跨设备提醒问题时，需要同时检查这台设备的系统通知权限。

**收集箱移动提示**

任务从收集箱移动到其他位置时，是否给出"操作成功"的简短反馈。

关闭提示不会取消操作本身，也不会隐藏错误提示——只是去掉那个绿色的"已完成"小通知。
