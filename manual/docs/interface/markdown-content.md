GranoFlow 的长文本字段使用富文本编辑。你可以直接输入文字、换行、加粗、插入链接，也可以用工具栏添加结构化内容。

常见位置包括任务描述、项目或里程碑说明、回顾记录，以及支持富文本的卡片字段。不同页面的入口文案可能不同，但编辑方式保持一致。

## 打开编辑器

在支持富文本的字段里点击描述或内容区域，会进入编辑页。编辑页默认就是所见即所得：你直接在正文里输入、选中文字加粗或斜体、插入链接，保存后再打开仍会按富文本显示。

顶部常用按钮包括：

| 按钮 | 用途 |
| --- | --- |
| 表格 | 插入可编辑表格 |
| 图片 | 从本机选择图片并作为附件插入 |
| 媒体 | 插入远程音频、视频或第三方视频链接，并可填写标题 |
| 公式 | 打开公式编辑弹窗，上方输入公式，下方预览效果 |
| 流程图 | 插入 Mermaid 流程图定义 |
| 保存 | 保存当前内容并返回上一页 |

在宽屏上，高级工具按钮会出现在编辑区上方；在手机或窄屏上，编辑器保留基础文字工具，避免工具栏占用正文空间。

## 添加表格

点击工具栏里的「表格」，GranoFlow 会插入一个可编辑表格。插入后直接点进单元格输入文字；需要换行时按 Enter，输入内容会保留在当前单元格或正文里。

表格适合记录对比、步骤、资料清单和复习卡片里的结构化知识。保存后再打开，表格仍会以表格形式显示和编辑。

## 添加公式

点击「公式」会打开专用弹窗。弹窗上方输入 LaTeX 公式，下方实时显示预览；确认后，公式会作为一个对象插入正文。

例如可以输入：

```text
\sum_{i=1}^{n} x_i = \frac{n(n+1)}{2}
```

公式对象在编辑区里显示为预览块。需要修改时，点击公式对象重新打开弹窗编辑。公式内容会随字段一起保存；学习卡片和阅读页面会显示公式预览，而不是把公式定义直接铺在正文里。

## 添加本地图片

点击「图片」，从本机选择图片。GranoFlow 会把图片作为当前内容的附件保存，并在正文里插入图片对象。

图片标题会用于说明和失败占位。图片文件缺失、解码失败或暂时无法读取时，页面会显示简短占位，不会把完整本地路径铺在正文里。

如果你想保存网页上的图片，先把图片下载到本机，再通过「图片」按钮插入。

## 添加远程音频

点击「媒体」，选择或填写音频链接，并为它写一个标题。渲染时会显示一个音频入口，点击后再打开或播放；页面不会在你点击前自动请求外部网站。

当前识别的常见音频后缀包括 `.mp3`、`.m4a`、`.aac`、`.wav`、`.ogg`。如果链接不是直接音频文件，而是一个播放网页，可能只能作为普通网页打开。

## 添加 YouTube 视频

点击「媒体」，填写 YouTube 链接并写一个标题。GranoFlow 不会在正文里自动加载第三方播放器；视频会以入口形式出现，点击后再打开外部页面或播放器。

需要登录、地区受限、年龄限制或已下架的视频无法保证可播放。建议在标题或正文里写清视频内容，避免外部链接失效后看不出原意。

## 流程图

点击「流程图」会插入 Mermaid 流程图定义。保存后，正文和卡片练习页会直接显示流程图预览。流程图适合表达步骤、依赖关系和简单决策路径。

Mermaid 对缩进、箭头和节点字符比较敏感。流程图无法显示时，GranoFlow 会显示简短失败态。需要修改时，回到编辑页点击流程图对象，调整定义后重新保存；你也可以把流程图定义交给 AI 修复后再粘贴回来。

## 远程资源确认和设置

远程音频、视频、YouTube 或 Vimeo 链接都可能向外部服务发起请求。GranoFlow 的原则是：没有确认或点击前，不自动加载远程内容。

第一次点击或加载远程资源时，你会看到确认弹窗。勾选“以后不再提示”后，当前设备会记住这个选择。这个设置只影响当前设备，不会改变你的任务、项目、回顾或卡片内容。

想重新看到确认弹窗时，进入「设置」→「Pro 设置」，找到「加载远程 Markdown 资源前提醒」，把它改回需要提醒的状态。开关关闭提醒后，远程图片、音频和视频会在允许范围内直接加载；重新开启后，下次加载远程资源前会再次询问。

## 排查显示问题

如果内容显示不正常，先回到编辑页检查对应对象：

- 表格不显示：确认表格仍有表头和单元格内容。
- 公式不显示：点击公式对象，检查输入的 LaTeX 是否完整。
- 图片不显示：确认本地附件仍可读取，或重新插入图片。
- 音频或视频打不开：确认链接仍可访问，且不需要额外登录。
- YouTube 打不开：确认视频没有下架、地区限制或登录要求。
- 流程图不显示：回到编辑页点击流程图对象，检查 Mermaid 定义是否完整。

如果内容很重要，建议在媒体或流程图旁补一句文字说明。外部媒体可能失效，但文字说明会继续留在 GranoFlow 里。
