GranoFlow 的主界面很克制，你不会一打开就被一堆按钮淹没。

底部（横屏或桌面时在左侧）有四个入口和一个快速新增按钮——这就是整个导航。

![界面导航总览](../../screenshots/zh-cn/interface-overview-navigation.png)

![底部导航根路由](../../screenshots/zh-cn/interface-root-route.png)

## 四个主要入口

| 入口 | 在这里能做什么 |
|------|--------------|
| **进展** | 默认首页，看今天完成了什么、项目在哪个阶段 |
| **任务** | 查看和推进任务清单 |
| **回顾** | 日回顾、周回顾、月度日历 |
| **设置** | 账号、同步、数据、AI 辅助、外观与偏好 |

## 快速新增

底部中间那个 **+** 是快速新增入口。

点一下，写下一件事，可以直接设置日期、项目、里程碑和标签，也可以什么都不设置——先写进去，之后再整理。

它不会切换到一个单独页面，你随时都能从当前界面快速捕捉一个想法。

:::tip[找不到某个功能？]
先找设置 ——大多数不常用的功能入口都在设置里。任务、项目、回顾这些主要功能直接在底部导航切换。
:::
