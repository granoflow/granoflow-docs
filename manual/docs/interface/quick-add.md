---
title: "快速添加"
description: "快速添加用于快速记下一件事，并可同时设置日期、提醒、标签、项目或里程碑。"
---

快速添加就是用来马上记下一件事：点击底部或侧边的 `+`，输入任务标题，再提交即可保存。只写标题也可以；如果没有设置日期、项目或里程碑，这条任务会先进入收集箱，之后再整理。

## 直接写标题

你可以把输入框当成一张便签，直接写一行普通文字。例如：

```text
整理周报
给 Alex 回邮件
检查发布截图
```

写完后点击提交按钮。GranoFlow 会把这一行保存为任务标题。

## 在标题里加字段

桌面端输入框会提示 `#标签 @项目 ~提醒`。这些符号只是快捷入口，不是必填项。

![快速添加 界面截图](../../screenshots/zh-cn/interface-quick-add-main.png)

- 输入 `#`，可以搜索标签。
- 输入 `@`，可以搜索项目或里程碑。
- 输入 `~`，可以填写提醒时间。

例如：

```text
检查订阅页文案 @官网改版 #发布 ~明天 8点
```

只有在你选择候选项，或用 `Enter` / `Tab` 确认后，快捷内容才会写入任务字段。没有确认的 `#发布` 或 `@官网改版` 会留在标题里，按普通文字保存。

## 日期和提醒

你也可以直接在标题里写日期词。例如：

```text
周五前检查发布截图
```

日期词会先被高亮，或显示成待确认的日期提示。点击日期提示，或在日期词后输入空格完成确认后，它才会成为任务日期。没有确认的日期词会继续留在标题里。

提醒用 `~` 开始。例如：

```text
明天整理截图 ~8点
给 Alex 回邮件 ~8am
```

提醒是通知时间，不等同于任务日期。如果当前任务还没有日期，GranoFlow 会根据提醒时间补一个合适的任务日期；你也可以用输入框下方的日期按钮手动选择。

## 用下方按钮选择

如果不想记快捷符号，可以直接用输入框下方的按钮：

- 日期
- 提醒
- 加入项目
- 标签
- 图片

这些按钮和 `#`、`@`、`~` 写入的是同一组任务字段。已经选择的字段会显示成小标签；你可以再次点击修改，也可以移除它。  
图片按钮每次最多添加 5 张，添加后会显示 `图片 N/5`。提交任务后，图片会跟随这条任务上传；如果网络或权限导致部分图片上传失败，任务仍会保存，界面会提示你“任务已创建，图片没有全部上传”。

如果选中的自定义标签带有模板，快速添加会先创建任务，再在后台把模板描述和根节点复制进去。模板添加失败不会撤销这条任务。

## 建议和纠错

输入时，GranoFlow 可能会显示相似任务建议。点击建议后，GranoFlow 会套用那条任务的标题，以及它最近一次保存的标签、项目或里程碑，并直接创建一条新任务。

如果系统发现明显拼写问题，第一次提交可能会先显示修正后的文字，而不是立刻保存。检查修正后的标题后，再次提交即可保存。

## 移动端和桌面端

移动端的输入提示更短，通常只提示你输入新任务。桌面端会显示 `#标签 @项目 ~提醒`，方便用键盘快速补字段。

不管在哪个端，快捷符号都只是加速方式。你也可以完全不用它们，只通过下方按钮设置日期、提醒、项目、里程碑和标签。
