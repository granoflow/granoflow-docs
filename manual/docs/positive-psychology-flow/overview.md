如果你只想先记住一句话：**第三章不是教你“每天都要心流”，而是教你在现实生活里更容易开始、持续、回到方向。**

这章只做一件事：把积极心理学与心流翻译成 GranoFlow 里可执行的路径。

## 这章在解决什么

你可能会遇到这些情况：

- 知道某件事重要，却总在等状态变好再开始
- 任务很多，但看不出和长期方向的关系
- 中断几天后，不知道从哪一步回来

第三章对应的答案是：

> 先看方向（价值）→ 再选容器（项目/里程碑）→ 再做一步（任务）→ 再留反馈（回顾）

## 第三章阅读地图（页面角色）

这一章分成五页，每页只负责一个主问题：

| 页面 | 只回答什么 |
| --- | --- |
| 快速开始 | 我现在就能做的一轮最小实践是什么？ |
| 核心概念 | 领域/价值观/项目/里程碑/任务/回顾分别是什么意思？ |
| 把价值变成可投入任务 | 为什么“重视”必须落成今天能开始的一步？ |
| 稳定投入 | 中断后如何回来，如何保护长期投入？ |
| 本页（总览） | 全章主线是什么，先读哪页最合适？ |

如果你不确定先读哪一页，先从「快速开始」开始，再回来看「核心概念」。

## GranoFlow 在这章的立场

- 不承诺你一定进入心流。
- 不要求你先把状态调整完美。
- 不把中断视为失败。

它提供的是一条可执行路径：

1. 用价值回答“为什么值得做”。
2. 用项目与里程碑回答“这一段先推进什么”。
3. 用任务回答“现在先做哪一步”。
4. 用回顾回答“这一步说明了什么，下一步是什么”。

## 这一章怎么用最省力

- 第一次使用：读「快速开始」，先跑一轮。
- 卡在概念：读「核心概念」。
- 知道重要但动不了：读「把价值变成可投入任务」。
- 经常中断：读「稳定投入」。

你不需要一次读完全部。按你当前卡点读一页就够。
