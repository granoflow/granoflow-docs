---

如果你想管理一个较大的目标，比如一次发布、一门课程或一段时间的计划，就可以创建项目：进入项目页，点击右上角 **+**，填写项目名称和必要说明，然后创建。

## 创建入口

在项目列表页右上角点击 **+** 按钮，会弹出创建对话框。

![项目列表入口](../../screenshots/zh-cn/projects-overview-main.png)

创建时可以填写：

- **名称**（必填）：写清楚这个项目要完成什么。比如“Q3 产品发布”比“项目一”更容易看懂。
- **描述**（可选）：记录背景、范围、约束或交付说明。之后打开项目时，它会作为项目上下文保留下来。
- **领域**（可选）：把项目放到一个大方向下，比如“工作”“学习”“健康”。

:::tip[名字起好，以后省事]
项目名称会出现在项目列表里。建议用“动词 + 目标”的写法，比如“写完毕业论文”，不要只写“论文”。
:::

## 创建空项目 vs 先建项目再加任务

两种方式都可以，按你的使用习惯选就行：

| 方式 | 适合场景 |
| --- | --- |
| 先建项目，再慢慢加任务 | 还在规划阶段，想先把项目框架建起来 |
| 先把任务加到收集箱，再归入项目 | 正在执行，想先快速记下来，之后再整理 |

不管用哪种方式，最后任务都可以放进项目里管理。

## 项目建好之后

项目创建后，你通常可以继续做这些事：

1. 添加里程碑（可选）——如果这个目标有明显阶段。
2. 把已有任务连接进项目和某个里程碑。
3. 在里程碑下直接创建新任务。

如果你现在还不知道要加哪些任务，也可以先关闭项目页面。项目已经创建好，之后再回来补充就行。
