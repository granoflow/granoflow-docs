任务可以挂在项目下，也可以进一步挂在某个里程碑下。挂进去之后，任务不会从其他视图消失，它会同时出现在项目视图和任何基于日期的视图里。

## 怎么连接

有两种方式：

**方法一：在任务详情里选择项目**

打开任务，在项目字段里选择目标项目（和里程碑）。这是最常用的方式。

**方法二：在项目页面拖入任务**

在项目详情页里，直接拖拽现有任务到里程碑下。

![拖拽把任务连接到项目](../../screenshots/zh-cn/projects-link-tasks-drag.png)

## 任务连接后出现在哪里

连接项目之后，同一个任务会同时出现在：

| 视图 | 说明 |
|------|------|
| 项目页面 | 出现在对应里程碑下 |
| 今日视图 | 如果任务有今天的日期，仍然出现 |
| 日历视图 | 按截止日期显示 |
| 收集箱 | **不再出现**（挂上项目后会离开收集箱）|

:::note[收集箱的变化]
只有同时满足"无截止日期""无项目""无里程碑"的任务，才会待在收集箱。一旦你把任务挂进项目，它就自动离开收集箱，这是正常行为。
:::

## 想把一个任务从项目里移出来

在任务详情里，把项目字段清空就行。清空后，如果任务也没有截止日期，它会重新出现在收集箱。

## 一个任务能挂多个项目吗

不能。每个任务只能属于一个项目（和一个里程碑）。

如果你发现有个任务想跨两个项目，通常有两种处理方式：

1. 选择任务更主要归属的项目
2. 把它拆成两个任务，分别挂到各自的项目
