---
title: "自动摆布任务时间"
description: "当日回顾里多条任务时间重叠时，可以把当天任务重新排成一条更容易检查的时间线。"
---

如果你经常在忙完之后才补任务，日回顾里可能会出现好几条任务挤在同一个时间点。自动摆布任务时间，就是先把这些记录放进一条更接近当天节奏的预览时间线，方便你回看、拖动微调，再决定是否保存。

这不是日历排班，也不是证明真实发生时间的工具。任务时间在 GranoFlow 里仍然可以重叠；这个入口只是给事后回顾一个更轻的整理入口。

## 从哪里开始

打开日回顾。如果右侧栏提示任务时间有重叠，你会看到“自动摆布任务时间”的入口。
没有重叠任务时，这条提示不会显示；你仍然可以直接使用“回顾今日任务”。

## 自动摆布会怎么做

第一版默认按轻量规则处理：

| 规则 | 含义 |
| --- | --- |
| 开始时间 | 当天 9:00 |
| 结束时间 | 你点击排布时的当前时间 |
| 排布方式 | 按任务顺序均匀分配 |
| 时间线 | 在预览弹窗里可以重叠，任务之间也可以留空隙 |
| 底层规则 | 不改变任务时间允许重叠的合法性 |

任务顺序会优先参考完成顺序、创建顺序和当前显示顺序。点击入口后不会直接改任务时间；GranoFlow 会先打开底部预览弹窗，你可以检查结果是否合理，再保存。
## 排布后怎么检查

预览弹窗旁边也有时间轴。你可以拖动任务块上沿改开始时间，拖动下沿改结束时间；任务块不能横向拖动。桌面端也可以直接在任务块里输入开始和结束时间，再用 Tab 键切换到下一个输入框。

重点看三件事：

1. 总时间窗口是否接近你今天实际投入的范围。
2. 任务顺序是否大体符合你当天做事的先后。
3. 单个任务的时间是否明显过长或过短。

如果只是为了让日回顾更容易看，不需要把每分钟都调到绝对准确。你只要把明显挤在一起的记录整理开，就已经足够有用。

预览弹窗里允许任务时间重叠，这样你可以直接把某一项拖到或输入到更贴近实际的时间段；保存前 GranoFlow 仍会检查开始时间早于结束时间，并且开始时间不能早于任务创建时间。

## 和回顾今日任务的关系

“自动摆布任务时间”先帮你整理时间块。“回顾今日任务”则让 AI 和你一起检查每个任务的耗时是否合理，并在确认后写回开始时间、结束时间、标题或任务回顾。

如果你不确定怎么分配时间，可以先自动摆布，再使用[回顾今日任务](../ai-assistance/daily-task-review)继续细化。
