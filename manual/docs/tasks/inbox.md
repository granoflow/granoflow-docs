收集箱用来临时放那些你刚想到、还没来得及整理的任务：先点 **+** 记下来，等有空再给它加日期、放进项目，或删掉不需要的事。

你可以把收集箱当成一个「先记再说」的地方。它不是最终的任务列表，也不是项目管理区。它只帮你先把事情从脑子里放下来，避免忘记。

## 怎么把任务放进收集箱

点底部导航栏中间的 **+**，写下任务，然后保存。只要你没有给它选择日期，它就会进入收集箱。

## 为什么有些任务没出现在收集箱

收集箱不会显示所有任务。它只显示**没有日期**，并且状态是待办或进行中的任务。项目和里程碑只是归属；一个没有日期的项目任务仍然可以出现在收集箱里。

| 你当时做了什么 | 任务去哪了 |
| --- | --- |
| 没选任何东西，直接保存 | 进收集箱 ✅ |
| 选了一个日期 | 出现在那一天的任务列表 |
| 加入了某个项目，但没选日期 | 进收集箱，同时归属到项目 |
| 在项目页面里创建，但没选日期 | 进收集箱，同时归属到那个项目 |

所以，任务没出现在收集箱，不代表它丢了。通常只是因为它已经有日期，或已经完成、归档、删除。

## 为什么任务会从收集箱里消失

只要任务有了日期，它就会离开收集箱。完成、归档或删除也会让它从收集箱消失；单独加入项目不会让没有日期的任务离开收集箱。

收集箱不是文件夹。它看的是任务有没有日期；项目或里程碑只说明这条任务属于哪里。

:::note[没看到的任务在哪？]
去“任务列表”（有日期的任务）、已完成/已归档页面，或具体的项目页面里找。整理后的任务会出现在对应视图中。
:::

## 怎么整理收集箱

你不需要每天清空收集箱，但重要的事不要放太久。每天开始时扫一眼，按下面的问题处理：

1. 这件事需要在某天完成吗？→ 给它加日期
2. 它属于某个项目吗？→ 移进对应项目
3. 它还值得做吗？→ 不值得就删除
4. 它现在很快就能做完吗？→ 三分钟内能完成的事，可以直接做掉

:::tip[一个好习惯]
每天早上花 2 分钟看一遍收集箱。把重要任务加上日期或项目，剩下暂时不急的可以先留着，不必强迫自己全部清空。
:::

这样用，收集箱就是一个帮你快速记录、之后再整理的入口，而不是一个让你焦虑的待处理堆。
