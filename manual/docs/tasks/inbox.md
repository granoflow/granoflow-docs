有什么事冒出来了？先记下来，别想那么多。

收集箱（Inbox）就是干这个用的——它是个临时停靠区，专门放那些"你还不知道放哪里"的任务。等有空了，再一件件整理好。

把它想成一个永远不会满的收件箱。不是你的工作场所，只是你的缓冲带。

## 怎么把任务放进收集箱

点底部导航栏中间的 **+**，写下任务，保存，就进收集箱了。

不需要选日期，不需要选项目，不需要加标签。越简单越好：

![收集箱界面](../../screenshots/zh-cn/tasks-inbox-main.png)

- "明天联系客户"
- "整理课程笔记"
- "买牙线"
- "周末约朋友吃饭"

第一步只是把它从脑子里"卸载"出来。细节等你有空再填。

## 为什么有些任务没出现在收集箱

收集箱不是放所有任务的地方，它只显示**没有日期、没有项目、没有里程碑**的待办或进行中任务。

| 你当时做了什么 | 任务去哪了 |
| --- | --- |
| 没选任何东西，直接保存 | 进收集箱 ✅ |
| 选了一个日期 | 出现在那一天的任务列表 |
| 加入了某个项目 | 出现在项目里 |
| 在项目页面里创建 | 自动归属到那个项目 |

任务没在收集箱不代表它丢了，只是它已经有了归属。

## 为什么任务会从收集箱里消失

给任务加了日期、加入了项目、完成了、归档了、删除了——只要满足其中一个，任务就会自动离开收集箱。

收集箱不是文件夹，整理好的任务不会留在这里。

:::note[没看到的任务在哪？]
去"任务列表"（有日期的）或具体的项目页面里找——那里才是它整理后的新家。
:::

## 怎么整理收集箱

不需要每天清空，但别让重要的事在这里放太久。每天开始时扫一眼，问自己：

1. 这件事需要在某天完成吗？→ 给它加个日期
2. 它属于某个项目吗？→ 移进对应项目
3. 它还值得做吗？→ 不值得就删掉
4. 它现在就可以做完吗？→ 三分钟内的事，直接做

:::tip[一个好习惯]
每天早上花 2 分钟扫一遍收集箱，把真正重要的任务给出日期或项目。剩下的先放着，不用强迫自己全清空。
:::

这样你的收集箱就是一个"先记后整"的工具，而不是一个让你焦虑的待处理堆。
