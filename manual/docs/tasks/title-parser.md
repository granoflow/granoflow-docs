如果你想快速给任务加标签、截止日期、项目归属或提醒，可以直接写在快速添加任务的标题里。比如输入 `写报告 明天 #工作 @论文初稿 ~下午3点`，GranoFlow 会把日期、标签、项目或里程碑，以及提醒显示成候选项；你确认后，它们才会写入对应字段。

## 怎么操作

在快速添加任务时，直接输入这些内容：

- `#工作`：候选添加一个叫“工作”的标签
- `@论文初稿`：候选关联一个项目或里程碑
- `明天`、`下周五`、`3月5日`：候选设置截止日期
- `~下午3点`：候选设置提醒时间

只要 GranoFlow 识别到 `#`、`@`、`~` 或日期词，输入框里会显示对应候选项。

标签、项目、里程碑和提醒候选项可以这样确认：

- 点击高亮区域里的候选项
- 按 **Enter** 或 **Tab** 确认当前候选项

日期候选也需要确认。你可以点击日期候选；如果刚输入了清楚的日期词，也可以在日期词后按空格，让 GranoFlow 把它写入截止日期字段。

:::caution[重要：确认前不会写入]
识别到的内容**不会**自动变成标签、日期、项目、里程碑或提醒。你必须明确确认，才会写入对应字段。没有确认的部分会继续留在标题文字里，不会偷偷消失。
:::

项目、里程碑和标签候选也遵守同一条规则：候选列表只帮你选择，真正写入字段要等你点击候选、按 Enter 或按 Tab 确认。按 Escape 关闭候选，或输入后没有确认，文字会继续作为普通标题保留。

## 常用语法速查

| 你输入的 | 效果 |
| --- | --- |
| `#工作` | 候选添加“工作”标签 |
| `@论文初稿` `@官网改版` | 候选关联项目或里程碑 |
| `明天` `下周五` `3月5日` | 候选设置截止日期 |
| `~下午3点` `~明早9点` | 候选设置提醒时间 |
| `写报告 明天 #工作 @论文初稿` | 同时候选日期、标签和项目或里程碑 |

## 有什么限制

- `#`、`@`、`~` 最好放在清楚的词边界前，比如 `#工作`，不要写成 `搞#工作`
- `@` 只用于项目或里程碑，不用于日期。要设置日期，直接写 `明天`、`下周五` 这类日期词
- 解析结果只是建议，最终以你确认后的字段为准
- 复杂句子里夹杂的符号，不一定都能被正确识别

如果某次识别结果不对，直接在字段里手动设置就好。自然语言输入只是快捷方式，不是必须使用的功能。
