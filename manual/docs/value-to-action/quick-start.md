title: "快速开始"
description: "用 5 分钟在 GranoFlow 完成第一次价值到行动实践：记录、整理、推进和回顾，不必先搭好完整人生计划。"
---

如果你第一次打开 GranoFlow，不需要先整理完整人生计划；先按底部中间的 `+`，写下一件现在真的要处理的事，然后决定它要不要变成任务、项目、里程碑或领域的一部分，晚上再做一次简短回顾就够了。

这一页会带你用最小的一轮，完成「记录、整理、推进、回顾」。它借用了 ACT（接纳与承诺疗法）和《幸福的陷阱》里「按价值行动」的思路，但你不需要先懂理论。你只要先把一件真实的事放进 GranoFlow，再让它慢慢变清楚。

## 先写下一件事

打开 GranoFlow 后，点击底部栏中间的 `+`。

写下你现在最明确的一件事。它可以很小，例如：

- 今天要回复的一封邮件
- 想继续推进的一项工作
- 一个突然想到但暂时不想忘记的想法

这里不需要写得完美，也不需要先决定它属于哪个项目。快速新增入口主要是用来「先收集」：先把事情从脑中放下来，再决定它应该去哪。

如果你暂时没有设置日期，也没有选择项目，它会先留在收集箱。收集箱不是失败区，也不是杂物间；它只是用来暂存还没有整理好的事情。

## 决定它属于哪里

保存之后，问自己一个简单问题：这件事只是一次性任务，还是属于更长期的目标？

如果它只是一次性事务，就把它当作普通任务处理。完成后勾选完成，它之后会成为你的历史记录。

如果它属于一个持续目标，可以把它连接到项目。项目适合承载一段时间内持续推进的目标，例如一次课程、一个产品版本、一段创作计划。

如果这个项目需要分阶段推进，可以继续添加里程碑。里程碑用来标记阶段目标，不是为了把项目变复杂，而是让你知道当前应该先完成哪一段。

如果它背后反复出现同一种长期方向，例如工作、学习、关系、健康或创作生活，可以再把它放到对应领域中。领域更接近你长期在意的方向，不需要第一次使用时就全部建好。

## 今天只推进一步

第一次使用时，不要急着整理所有任务、所有项目、所有人生方向。

先选择一个今天可以推进的动作，然后去完成它。

GranoFlow 的结构不是为了让计划变得更庞大，而是为了让下一步更清楚。一个任务太大时，可以拆成更小的步骤；只要下一步能开始，它就已经足够有用。

如果你把某个任务标记为进行中，系统会尽量让当前只有一个任务占用你的注意力。这样你不必同时盯着很多「应该做」的事，只需要知道现在先做哪一个。

完成后，把它标记为已完成。暂时不再需要日常关注、但仍然值得保留的内容，可以归档。误加或太早添加的内容，可以删除，并之后在回收站中恢复或继续清理。

## 晚上做一次简短回顾

当天结束时，打开「回顾」。

你不需要写很长。只要确认三件事：

- 今天完成了什么
- 有什么值得留下
- 下一步是什么

回顾不是给自己打分，也不是检查你有没有足够努力。它的作用是把一天中发生过的行动、选择和经验留下来，让它们不只是过去，而是能慢慢变成积累。

如果今天没有完成很多，也可以照常回顾。GranoFlow 不会因为中断清零；你随时停下，也可以随时回来。
