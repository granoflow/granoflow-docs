如果你想知道「刪除帳戶後資料會點」，答案係：雲端資料會喺寬限期後永久刪除，訂閱權益會失效；但每部裝置入面的本地 GranoFlow 資料唔會自動消失，需要你自己處理。

## 刪帳戶會發生咩事

| 項目 | 刪帳戶後 |
| --- | --- |
| 雲端資料 | 喺寬限期後永久刪除 |
| 各裝置本地資料 | **唔會自動刪除** |
| 訂閱權益 | 帳戶刪除後權益失效 |
| 可以撤銷嗎 | 寬限期內可以取消，之後不可逆 |

## 刪帳戶前的檢查清單

- [ ] 確認你唔再需要雲端同步或多裝置共享
- [ ] 如果你仲想保留資料，先匯出本地備份
- [ ] 如果你有進行中的訂閱，先了解退款政策
- [ ] 如果你有多部裝置，先確認每部裝置上仲有邊啲本地資料

## 刪帳戶 vs 登出

- **登出**：只係斷開呢部裝置同帳戶的連接。本地資料仍然保留，之後可以重新登入。
- **刪除帳戶**：註銷帳戶。雲端資料會喺寬限期後刪除，寬限期結束後無法恢復。

如果你只係暫時唔想用 GranoFlow，通常登出就夠。刪除帳戶適合你確定唔再需要呢個帳戶同雲端資料的情況。

## 寬限期

提交帳戶刪除申請後，通常會有一段寬限期，實際時長請睇 App 內說明。喺寬限期內，你可以取消刪除申請，令帳戶恢復正常。寬限期結束後，刪除操作不可撤銷。

:::caution[本地資料唔會自動消失]
刪除帳戶後，你每部裝置上的本地 GranoFlow 資料仍然存在，直到你手動解除安裝 App 或清除 App 資料。如果你唔想保留呢啲本地資料，需要喺對應裝置上手動清除。
:::
