---

刪除 GranoFlow 賬號係一個不可逆操作。喺操作前，有幾件事需要搞清楚。

## 刪賬號會發生什麼

| | 刪賬號後 |
|--|---------|
| 雲端數據 | 喺寬限期後永久刪除 |
| 各設備本地數據 | **唔會自動刪除** |
| 訂閱權益 | 賬號刪除後權益失效 |
| 可以撤銷嗎 | 寬限期內可以取消，過後不可逆 |

## 刪賬號前的檢查清單

- [ ] 確認你唔再需要雲端同步或多設備共享
- [ ] 如果你還想保留數據，先導出本地備份
- [ ] 如果你有正在進行中的訂閱，了解退款政策
- [ ] 如果你有多台設備，了解每台設備上的數據狀態

## 刪賬號 vs 退出登錄

- **退出登錄**：斷開呢台設備和賬號的連接，本地數據保留，可以重新登錄
- **刪除賬號**：注銷賬號，雲端數據喺寬限期後刪除，不可恢復

如果你只係暫時唔想用 GranoFlow，退出登錄就夠了。刪賬號係永久操作。

## 寬限期

賬號刪除申請提交後，通常有一段寬限期（具體時長見 App 內說明）。寬限期內你可以取消刪除申請，讓賬號恢復正常。寬限期結束後，刪除操作不可撤銷。

:::caution[本地數據唔會自動消失]
刪除賬號後，你每台設備上的本地 GranoFlow 數據仍然存在，直到你手動卸載 App 或清除 App 數據。如果你唔想要呢些本地數據，需要手動清除。
:::
