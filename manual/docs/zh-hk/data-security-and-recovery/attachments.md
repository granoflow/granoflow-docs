如果你想知圖片係咪會同任務一齊安全保存，先記住一句：文字任務同圖片附件係分開儲存、分開同步。文字可能已經去到另一部裝置，但圖片仲上傳緊；「清空本地附件（保留圖片）」只會清走呢部裝置入面的非圖片附件，唔會用呢個入口清走圖片。

![附件管理入口](../../../screenshots/zh-hk/subscription-vip-settings.png)

## 圖片同文字有咩唔同

- **文字任務**：內容細，同步快，通常好快就可以喺本地同雲端保持一致
- **圖片附件**：檔案大，同步慢，可能喺文字任務已經同步之後都仲未傳完

所以，你喺另一部裝置見到任務已經出現，但圖片暫時未顯示，唔一定代表任務遺失。通常先保持網絡穩定，再俾圖片多啲時間上傳同下載。

## 刪除圖片附件代表咩

![附件與圖片 界面截圖](../../../screenshots/zh-hk/data-attachments-clear-detail.png)

處理圖片相關內容時，要分清兩件事：

- **從任務移除**：只係令呢個任務唔再連住張圖片；呢部裝置上的本地檔案可能仍然存在
- **清空本地附件（保留圖片）**：由 `設定 → Pro 設定` 進入，未開啟全量同步附件時可清空呢部裝置上的非圖片附件，用來釋放本地儲存空間；開啟全量同步附件時，呢個按鈕會停用

清空本地附件之後，被清走的非圖片附件之後通常可以重新下載。圖片係咪可以喺其他裝置見到，仍然取決於圖片之前有冇成功上傳到雲端；如果張圖從來未成功上傳，換裝置或者清除 App 本地資料之後就冇可恢復的來源。

## 本地備份會唔會包含圖片

本地備份會打包當前裝置上已有的業務記錄同附件檔案，包括已經落喺呢部裝置上的圖片、PDF 同通用附件。但佢唔會替你由雲端補齊呢部裝置尚未下載的附件；如果圖片或附件仲未上傳成功，或者雲端附件仲未落到本機，備份入面都可能缺少佢哋。如果你需要附件盡量完整，重點係要令雲端同步處於正常狀態；Pro 用戶仲應該先開啟「全量同步附件」，等同步完成之後再建立備份。

:::note[圖片上傳需要網絡]
圖片唔會離線上傳。例如你喺地鐵入面影咗張相並附到任務上，呢張圖片要等到下次有網絡時，先會繼續上傳到雲端。
:::
