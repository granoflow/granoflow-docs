GranoFlow 的雲端同步數據係端對端加密的。加密密鑰就像保險箱的鑰匙——**冇咗佢，連 GranoFlow 自己的服務器也無法讀取你的數據**。

但呢把鑰匙也意味著：**如果你自己丟咗，就真的搵唔番。**

![加密與恢復密鑰界面](../../../screenshots/zh-hk/data-encryption-recovery-key.png)

## 密鑰 vs 登錄密碼，有什麼區別

| | 登錄密碼（或驗證郵件） | 加密/同步密鑰 |
|--|----------------------|--------------|
| 用嚟做什麼 | 證明你係誰 | 打開雲端加密數據 |
| 忘咗點辦 | 重新發驗證郵件 | **無法找回** |
| 換咗會怎樣 | 只影響登錄 | 影響係否能訪問雲端數據 |

## 我的密鑰喺哪裏

喺 GranoFlow 設定 → 數據/安全/同步 裏可以查看和保存當前設備的雲端同步密鑰。

**強烈建議：把密鑰抄低，或保存到你的密碼管理器裏。**

## 喺新設備上需要密鑰的情況

當你：

- 換咗手機或電腦
- 重裝咗 GranoFlow
- 出現咗「同步密鑰唔匹配」的提示

GranoFlow 會要求你輸入舊設備上的雲端同步密鑰，先能訪問雲端的已有數據。

## 輸入密鑰後會發生什麼

GranoFlow 先驗證密鑰能唔能打開當前雲端數據：

- **密鑰匹配，雲端和本地係同一份數據** → 直接連接同步
- **密鑰匹配，但本地有新數據** → 顯示選擇界面，讓你決定保留哪份數據
- **密鑰唔對** → 唔會改變任何數據，讓你重新輸入

## 忘咗密鑰點辦

先唔好慌，按順序檢查：

1. **舊設備還能用嗎？** → 喺舊設備上打開 GranoFlow，搵到密鑰並複製
2. **密碼管理器裏有嗎？** → 檢查你的密碼管理器
3. **舊設備有但打唔開 App？** → 聯繫 GranoFlow 支持

如果以上都冇，雲端加密數據可能無法恢復。本地備份（如果有的話）仍然可用。

## 「用雲端密鑰替換本地密鑰」係什麼意思

呢個選項用於：當前設備有一份本地數據，但你決定讓呢台設備切換到使用雲端的加密方案。

**操作前務必確認**：你有當前雲端數據對應的完整密鑰，以及你清楚本機和雲端哪份數據更重要。

:::caution[密鑰唔係密碼，唔能重置]
加密密鑰丟失後，GranoFlow 無法幫你重置或找回。而家就去保存你的密鑰，唔好等到用到的時候先後悔。
:::
