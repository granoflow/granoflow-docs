多端同步的意思是：你在一部設備新增、修改或刪除內容之後，其他登入同一賬號的設備，都會盡量變成同一個狀態。它方便你在多部設備繼續使用 GranoFlow，但它不是用來救回誤刪內容的備份功能。

![同步狀態界面](../../../screenshots/zh-hk/data-sync-status-main.png)

## 同步甚麼、不同步甚麼

✅ 會同步的內容：

- 任務，例如標題、截止日期、標籤、狀態等
- 項目和里程碑
- 回顧記錄
- 圖片和附件，在網絡許可的情況下同步

⚠️ 最重要要記住：同步不是備份。

- **你刪除，其他設備也會刪除**：同步是雙向的，刪除操作不會只留在一部設備上。
- **沒有版本歷史**：同步不會保存「3 日前是甚麼樣」。
- **圖片可能遲一點出現**：文字內容可能先同步完成，圖片和附件可能之後才同步完成。

## 同步的常見狀態

| 狀態 | 含義 |
| --- | --- |
| 同步中 | 正在上傳或下載變更。 |
| 已同步 | 目前設備的數據和雲端一致。 |
| 等待中 | 有變更正在排隊，通常和網絡有關。 |
| 錯誤 | 同步遇到問題，需要檢查賬號或密鑰。 |

## 離線和服務無法連線時會怎樣

如果暫時沒有網絡，GranoFlow 內的本地數據仍然可以使用。你可以繼續記錄任務、整理項目、寫回顧、搜尋內容，也可以匯出本地備份。

如果你開啟了「離線模式」，同步、登入、訂閱確認和恢復購買會暫時不可用。你點按這些需要連線的功能時，App 會提示目前無法連接相關服務；關閉離線模式後，可以再試一次。

如果設備有網絡，但同步服務暫時無法連線，App 也不會阻止你使用本機已有的數據。已經在這部設備上的內容仍然可以編輯；需要上傳到雲端或從雲端下載的變更，會等你稍後再次同步。

會員裝置可以在「設定 > 訊息與提醒」中開啟靜默同步。靜默同步用於讓其他裝置的新資料自動補到本機，但它保持靜默：不會彈系統橫幅，不會播放聲音，也不會新增通知中心訊息。是否能開啟仍取決於登入狀態、會員權益和目前平台是否支援。

## 新設備加入同步

如果你換了新手機，或者重新安裝 App，想接上原有的雲端數據，需要使用舊設備上的**雲端同步密鑰**。

詳細步驟見 → [新設備同步已有雲端數據](/manual/zh-hk/data-security-and-recovery/new-device-sync/)

:::caution[同步不等於備份]
請定期匯出本地備份。誤刪的任務不能靠同步恢復，因為雲端和其他設備也會一起刪除。
:::
