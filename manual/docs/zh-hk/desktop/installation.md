---
title: "桌面端安裝"
description: "下載、安裝 GranoFlow 桌面端，以及首次打開時遇到系統攔截點算。"
---

如果你想喺電腦上用 GranoFlow，先去 GranoFlow 官網下載安裝包，安裝後用同一個帳號登入。桌面端可以留喺電腦上使用，亦可以用快捷鍵呼出，唔需要每次都先開瀏覽器。

## 下載安裝

打開 [GranoFlow 官網](https://granoflow.com)，下載適合你電腦系統嘅安裝包：

- **macOS**：下載 `.dmg` 文件，打開後將 GranoFlow 拖入「應用程式」文件夾。
- **Windows**：下載 `.exe` 安裝包，然後跟住安裝嚮導一步步完成安裝。

## macOS 提示「無法打開」點算

第一次打開 GranoFlow 時，macOS 可能會提示佢來自未知開發者，令你無法直接打開。

可以咁做：

1. 打開 Finder，搵到 GranoFlow 應用程式。
2. 右鍵點擊 GranoFlow。
3. 選擇「打開」。
4. 系統再次彈出確認視窗時，再點擊「打開」。

你亦可以喺系統設定處理：打開「系統設定」→「私隱與安全性」，搵到 GranoFlow 嘅攔截記錄，然後點擊「仍然打開」。

## Windows 提示 SmartScreen 點算

第一次執行 GranoFlow 時，Windows Defender SmartScreen 可能會彈出警告。

喺呢個提示入面，先點擊「更多資訊」，再點擊「仍要執行」。

## 安裝完成後

安裝完成後，打開 GranoFlow，並用你嘅帳號登入。如果呢個帳號之前已經有雲端數據，GranoFlow 會開始同步呢啲數據。

:::note[桌面端和手機端共用同一帳號]
喺桌面端同手機端登入同一個帳號後，數據會透過雲端同步，喺唔同裝置之間共享。
:::

如果你需要喺終端機調用 GranoFlow，下一步閱讀 [命令行工具](/manual/zh-hk/desktop/command-line-tool/)。
