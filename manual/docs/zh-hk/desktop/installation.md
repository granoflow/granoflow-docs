---
title: "桌面端安裝"
description: "下載、安裝 GranoFlow 桌面端，以及首次打開時遇到系統攔截點算。"
---

桌面端讓 GranoFlow 常駐在你的電腦上，隨時用快捷鍵呼出，唔需要每次打開瀏覽器。

## 下載安裝

從 [GranoFlow 官網](https://granoflow.com) 下載對應系統的安裝包：

- **macOS**：下載 `.dmg` 文件，打開後拖入應用程式文件夾
- **Windows**：下載 `.exe` 安裝包，按照安裝嚮導完成

## macOS 提示「無法打開」點算

第一次打開時，macOS 可能提示「來自未知開發者」。

解決方法：

1. 在 Finder 裡右鍵點擊應用 → 選擇「打開」
2. 再次點擊「打開」確認

或者：系統設定 → 私隱與安全性 → 找到 GranoFlow 的攔截記錄 → 點擊「仍然打開」。

## Windows 提示 SmartScreen 點算

Windows Defender SmartScreen 可能彈出警告。點擊「更多資訊」→「仍要執行」即可。

## 安裝完成後

首次打開後用你的帳號登入，GranoFlow 會自動開始同步雲端數據（如果你之前有雲端數據的話）。

:::note[桌面端和手機端共用同一帳號]
登入同一帳號，數據會透過雲端同步在桌面端和手機端之間共享。
:::
