如果你想用最短路徑開始使用 GranoFlow 這款個人規劃應用，可以從這頁開始。不用一開始就重建整個生活系統。

## 先捕捉一件事

打開快速添加，寫低腦中最清楚的一個任務或想法。第一分鐘可以把 GranoFlow 當成 Todo 應用：先記下來，再決定它應該放在哪裏。

## 有需要時再連接

如果這件事屬於更長方向，把它連接到項目或里程碑。這樣，一個日計劃動作就會成為目標規劃、項目規劃器和個人時間結構的一部分。

## 回顧，不給自己打分

一天結束前，望一望完成了甚麼，再寫下下一步。日回顧不是分數、連續打卡或羞辱機制，而是讓長期目標和複盤保持連接的安靜方式。
