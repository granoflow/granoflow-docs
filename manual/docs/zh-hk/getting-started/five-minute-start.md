title: "5 分鐘上手"
description: "先記一個任務，需要時連接到項目，再用一次簡短回顧結束今日。"
---

想最快開始用 GranoFlow，就先做三件事：新增一個任務，需要時把它連接到項目，今晚用一兩句話回顧今日。不需要先建立完整系統。

## 第一步：先寫下一個任務

打開 GranoFlow，點 **+**，寫下你而家腦中最清楚的一件事。

暫時不用判斷它重不重要，也不用急着決定它屬於哪個項目。第一步只是先記低。新任務會先進入收集箱，之後你有時間再整理。

![快速新增任務面板](../../../screenshots/zh-hk/interface-quick-add-main.png)

:::tip[快捷鍵]
桌面端可以用全局快捷鍵直接叫出快速新增，不用先切換到 GranoFlow 視窗。快捷鍵可以在設定入面自訂。
:::

## 第二步（可選）：需要時連接到項目

如果這件事屬於一個更大的方向，例如「完成論文」或「準備搬屋」，你可以在任務詳情入面把它連接到對應項目。

這一步不是必須。你可以先把任務留在收集箱，等項目結構真的幫到手時再整理。

## 第三步：今晚做一次小回顧

今日結束前，打開 GranoFlow，望一望今日完成了甚麼、還有甚麼未完成，然後在日回顧寫一兩句：

- 今日做了甚麼
- 明日最想推進哪一件事

不用寫長篇總結，不用打分，也不用變成每日必做的壓力。它只是幫你把今日、明日同長期目標輕輕連起來。
