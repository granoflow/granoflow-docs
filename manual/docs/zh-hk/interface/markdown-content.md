GranoFlow 的長文本欄位使用富文本編輯。你可以直接輸入文字、換行、加粗、插入連結，也可以用工具列加入結構化內容。

常見位置包括任務描述、項目或里程碑說明、回顧記錄，以及支援富文本的卡片欄位。不同頁面的入口文案可能不同，但編輯方式保持一致。

## 打開編輯器

在支援富文本的欄位裏點擊描述或內容區域，會進入編輯頁。編輯頁預設就是所見即所得：你直接在正文輸入、選中文字加粗或斜體、插入連結，儲存後再打開仍會按富文本顯示。

常用工具包括：

| 按鈕 | 用途 |
| --- | --- |
| 表格 | 插入可編輯表格 |
| 圖片 | 從本機選擇圖片並作為附件插入 |
| 媒體 | 插入遠程音頻、影片或第三方影片連結，並可填寫標題 |
| 公式 | 打開公式編輯彈窗，上方輸入公式，下方預覽效果 |
| 流程圖 | 插入 Mermaid 流程圖定義 |
| 儲存 | 儲存目前內容並返回上一頁 |

在寬屏上，高級工具會出現在編輯區上方；在手機或窄屏上，編輯器保留基礎文字工具，避免工具列佔住正文空間。

## 加入表格

點擊「表格」，GranoFlow 會插入一個可編輯表格。插入後直接點進儲存格輸入文字。表格適合記錄對比、步驟、資料清單和卡片裏的結構化知識。

{/* manual-screenshot:id=rich-content-table-insert */}

## 加入公式

點擊「公式」會打開專用彈窗。彈窗上方輸入 LaTeX 公式，下方即時顯示預覽。

例如可以輸入：

```text
\sum_{i=1}^{n} x_i = \frac{n(n+1)}{2}
```

公式會作為一個對象插入正文。要修改時，點擊公式對象重新打開彈窗。閱讀頁和卡片練習會顯示公式預覽，不會把公式定義直接鋪在正文裏。

{/* manual-screenshot:id=rich-content-formula-dialog */}

## 加入本地圖片

點擊「圖片」，從本機選擇圖片。GranoFlow 會把圖片作為目前內容的附件保存，並在正文裏插入圖片對象。

如果你想保存網頁上的圖片，先把圖片下載到本機，再用「圖片」插入。編輯器不會要求你貼外部圖片 URL。

{/* manual-screenshot:id=rich-content-local-image */}

## 加入遠程音頻

點擊「媒體」，填寫直接音頻連結，並為它寫一個標題。GranoFlow 會顯示一個音頻入口，等你點擊後才請求外部資源。

常見直接音頻後綴包括 `.mp3`、`.m4a`、`.aac`、`.wav`、`.ogg`。

{/* manual-screenshot:id=rich-content-remote-audio-trigger */}

## 加入 YouTube 影片

點擊「媒體」，貼上 YouTube 連結並寫一個標題。GranoFlow 不會在正文裏自動載入第三方播放器；它會先顯示影片入口，你需要時再打開。

需要登入、地區受限、年齡限制或已下架的影片不一定能播放。建議在旁邊補一句文字說明，外部連結失效後仍知道原意。

{/* manual-screenshot:id=rich-content-youtube-video-trigger */}

## 流程圖

點擊「流程圖」會插入 Mermaid 流程圖定義。流程圖適合表達步驟、依賴關係和簡單決策路徑。

流程圖無法顯示時，GranoFlow 會顯示簡短失敗狀態，並保留編輯入口。你可以自行修正，也可以交給 AI 修復後再貼回來。

{/* manual-screenshot:id=rich-content-mermaid-flow */}

## 遠程資源

遠程音頻、影片、YouTube 或 Vimeo 連結都可能向外部服務發出請求。GranoFlow 的原則是：沒有確認或點擊前，不自動載入遠程內容。

第一次載入遠程資源時，你可能會看到確認彈窗。選擇以後不再提示後，目前設備會記住這個選擇。

{/* manual-screenshot:id=rich-content-remote-resource-setting */}

## 排查顯示問題

- 表格不顯示：重新打開編輯器，檢查儲存格內容。
- 公式不顯示：點擊公式對象，檢查 LaTeX 是否完整。
- 圖片不顯示：確認本地附件仍可讀取，或重新插入圖片。
- 音頻或影片打不開：確認連結仍可存取，並且不需要額外登入。
- 流程圖打不開：點擊流程圖對象，檢查 Mermaid 定義是否完整。

如果內容很重要，建議在媒體或流程圖旁補一句文字說明。外部連結可能失效，但文字說明會繼續留在 GranoFlow 裏。
