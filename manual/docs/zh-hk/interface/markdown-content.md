GranoFlow 的長文字欄位使用豐富文字編輯。你可以直接輸入文字、換行、加粗、插入連結，也可以用工具列新增結構化內容。

常見位置包括任務描述、項目或里程碑說明、回顧記錄，以及支援豐富文字的卡片欄位。不同頁面的入口文案可能不同，但編輯方式保持一致。

## 開啟編輯器

在支援豐富文字的欄位裡點擊描述或內容區域，會進入編輯頁。編輯頁預設就是所見即所得：你直接在正文裡輸入、選中文字加粗或斜體、插入連結，儲存後再打開仍會按豐富文字顯示。

頂部常用按鈕包括：

| 按鈕 | 用途 |
| --- | --- |
| 表格 | 插入可編輯表格 |
| 圖片 | 從本機選擇圖片並作為附件插入 |
| 媒體 | 插入遠端音訊、影片或第三方影片連結，並可填寫標題 |
| 公式 | 開啟公式編輯彈窗，上方輸入公式，下方預覽效果 |
| 流程圖 | 插入 Mermaid 流程圖定義 |
| 儲存 | 儲存當前內容並返回上一頁 |

在寬螢幕上，進階工具按鈕會出現在編輯區上方；在手機或窄螢幕上，編輯器保留基礎文字工具，避免工具列佔用正文空間。

## 新增表格

點擊工具列裡的「表格」，GranoFlow 會插入一個可編輯表格。插入後直接點進儲存格輸入文字；需要換行時按 Enter，輸入內容會保留在當前儲存格或正文裡。

表格適合記錄對比、步驟、資料清單和複習卡片裡的結構化知識。儲存後再打開，表格仍會以表格形式顯示和編輯。

## 新增公式

點擊「公式」會開啟專用彈窗。彈窗上方輸入 LaTeX 公式，下方即時顯示預覽；確認後，公式會作為一個物件插入正文。

例如可以輸入：

```text
\sum_{i=1}^{n} x_i = \frac{n(n+1)}{2}
```

公式物件在編輯區裡顯示為預覽區塊。需要修改時，點擊公式物件重新開啟彈窗編輯。公式內容會隨欄位一起儲存；學習卡片和閱讀頁面會顯示公式預覽，而不是把公式定義直接鋪在正文裡。

## 新增本地圖片

點擊「圖片」，從本機選擇圖片。GranoFlow 會把圖片作為當前內容的附件儲存，並在正文裡插入圖片物件。

圖片標題會用於說明和失敗佔位。圖片檔案缺失、解碼失敗或暫時無法讀取時，頁面會顯示簡短佔位，不會把完整本地路徑鋪在正文裡。

如果你想儲存網頁上的圖片，先把圖片下載到本機，再透過「圖片」按鈕插入。

## 新增遠端音訊

點擊「媒體」，選擇或填寫音訊連結，並為它寫一個標題。呈現時會顯示一個音訊入口，點擊後再開啟或播放；頁面不會在你點擊前自動請求外部網站。

當前識別的常見音訊副檔名包括 `.mp3`、`.m4a`、`.aac`、`.wav`、`.ogg`。如果連結不是直接音訊檔案，而是一個播放網頁，可能只能作為普通網頁開啟。

## 新增 YouTube 影片

點擊「媒體」，填寫 YouTube 連結並寫一個標題。GranoFlow 不會在正文裡自動載入第三方播放器；影片會以入口形式出現，點擊後再開啟外部頁面或播放器。

需要登入、地區受限、年齡限制或已下架的影片無法保證可播放。建議在標題或正文裡寫清影片內容，避免外部連結失效後看不出原意。

## 流程圖

點擊「流程圖」會插入 Mermaid 流程圖定義。儲存後，正文和卡片練習頁會直接顯示流程圖預覽。流程圖適合表達步驟、依賴關係和簡單決策路徑。

Mermaid 對縮排、箭頭和節點字元比較敏感。流程圖無法顯示時，GranoFlow 會顯示簡短失敗態。需要修改時，回到編輯頁點擊流程圖物件，調整定義後重新儲存；你也可以把流程圖定義交給 AI 修復後再貼回來。

## 遠端資源確認和設定

遠端音訊、影片、YouTube 或 Vimeo 連結都可能向外部服務發起請求。GranoFlow 的原則是：沒有確認或點擊前，不自動載入遠端內容。

第一次點擊或載入遠端資源時，你會看到確認彈窗。勾選「以後不再提示」後，當前裝置會記住這個選擇。這個設定只影響當前裝置，不會改變你的任務、項目、回顧或卡片內容。

想重新看到確認彈窗時，進入「設定」→「Pro 設定」，找到「載入遠端 Markdown 資源前提醒」，把它改回需要提醒的狀態。開關關閉提醒後，遠端圖片、音訊和影片會在允許範圍內直接載入；重新開啟後，下次載入遠端資源前會再次詢問。

## 排查顯示問題

如果內容顯示不正常，先回到編輯頁檢查對應物件：

- 表格不顯示：確認表格仍有表頭和儲存格內容。
- 公式不顯示：點擊公式物件，檢查輸入的 LaTeX 是否完整。
- 圖片不顯示：確認本地附件仍可讀取，或重新插入圖片。
- 音訊或影片打不開：確認連結仍可存取，且不需要額外登入。
- YouTube 打不開：確認影片沒有下架、地區限制或登入要求。
- 流程圖不顯示：回到編輯頁點擊流程圖物件，檢查 Mermaid 定義是否完整。

如果內容很重要，建議在媒體或流程圖旁補一句文字說明。外部媒體可能失效，但文字說明會繼續留在 GranoFlow 裡。
