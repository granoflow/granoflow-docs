底部有四個入口和一個快速新增按鈕。

![界面導航總覽](../../../screenshots/zh-hk/interface-overview-navigation.png)

| 入口 | 用途 |
|------|------|
| **進展** | 默認首頁，看今天完成了什麼 |
| **任務** | 查看和推進任務 |
| **回顧** | 日回顧、週回顧、月度日曆 |
| **設置** | 帳號、同步、數據、外觀 |

底部中間的 **+** 是快速新增入口，隨時可以把一件事寫下來。
