如果你只想先記住一句話：**第三章不是教你「每天都要心流」，而是教你在真實生活中更容易開始、持續、回到方向。**

這章只做一件事：把積極心理學和心流翻譯成 GranoFlow 裏可執行的路徑。

## 這章在解決甚麼

你可能會遇到：

- 知道某件事重要，但總在等狀態變好再開始
- 任務很多，但看不出和長期方向的關係
- 中斷幾天後，不知道從哪一步回來

第三章對應的答案是：

> 先看方向（價值）→ 再選容器（項目/里程碑）→ 再做一步（任務）→ 再留反饋（回顧）

## 第三章閱讀地圖（頁面角色）

每一頁只回答一個主問題：

| 頁面 | 只回答甚麼 |
| --- | --- |
| 快速開始 | 我現在就能做的一輪最小實踐是甚麼？ |
| 核心概念 | 領域/價值觀/項目/里程碑/任務/回顧各自是甚麼？ |
| 把價值變成可投入任務 | 為甚麼「重視」必須落成今日可開始的一步？ |
| 穩定投入 | 中斷後如何回來，如何保住長期投入？ |
| 本頁（總覽） | 全章主線是甚麼，先讀哪頁最合適？ |

如果你不確定先讀哪一頁，先從「快速開始」開始，再回來看「核心概念」。

## GranoFlow 在這章的立場

- 不承諾你一定進入心流。
- 不要求你先把狀態調整到完美。
- 不把中斷視為失敗。

它提供的是一條可執行路徑：

1. 用價值回答「為何值得做」。
2. 用項目與里程碑回答「這段時間先推進甚麼」。
3. 用任務回答「現在先做哪一步」。
4. 用回顧回答「這一步說明了甚麼，下一步是甚麼」。

## 這章怎樣用最省力

- 第一次使用：讀「快速開始」，先跑一輪。
- 卡在概念：讀「核心概念」。
- 知道重要但動不了：讀「把價值變成可投入任務」。
- 經常中斷：讀「穩定投入」。

你不需要一次讀完全部，按你當下卡點讀一頁就夠。
