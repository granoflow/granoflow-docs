將任務連接到項目後，佢會出現喺項目裏面；如果佢有日期，亦會照樣出現喺今日視圖或日曆視圖。項目用嚟說明呢件事屬於邊個目標，里程碑用嚟說明佢屬於項目裏面嘅邊個階段。

呢度最容易誤會嘅係收集箱：**連接項目本身唔會令一個冇日期任務自動離開收集箱**。收集箱主要睇任務有冇日期；如果任務仍然冇日期，佢可能會繼續留喺收集箱，方便你稍後安排時間。

## 點樣連接

你可以用幾種方式將任務連接到項目。

### 方法一：喺任務詳情裏面揀項目

打開任務詳情，搵到項目欄位，揀目標項目。需要時，亦可以繼續揀呢個項目下嘅某個里程碑。

呢個係最常用嘅方式，適合你已經打開任務、想直接幫佢歸類嘅時候。

### 方法二：喺任務列表或收集箱裏面「加入項目」

喺任務列表或收集箱裏面用「加入項目」時，需要揀一個具體里程碑。咁樣任務會同時連接到項目同呢個階段。

### 方法三：喺項目頁面建立新任務

打開項目詳情頁，喺某個里程碑下新增任務。咁樣建立出嚟嘅任務會自動帶上當前項目同里程碑，唔需要再手動揀一次。

## 任務連接後出現喺邊度

連接項目之後，同一個任務可能會同時出現喺多個地方。佢唔係被複製咗，而係同一個任務喺唔同視圖裏面顯示。

| 視圖 | 說明 |
| --- | --- |
| 項目頁面 | 出現喺對應項目裏面；如果揀咗里程碑，會出現喺對應里程碑下 |
| 今日視圖 | 如果任務有今日嘅日期，仍然出現 |
| 日曆視圖 | 按截止日期顯示 |
| 收集箱 | 如果任務仍然冇日期，而且仲係待辦或進行中，可能會繼續出現 |

:::note[收集箱嘅變化]
收集箱唔係「冇項目嘅任務列表」，而係「仲未安排日期嘅待處理任務列表」。如果你將任務加入咗帶截止日期嘅里程碑，GranoFlow 可能會將里程碑日期帶到任務上；呢個時候佢會按日期進入任務列表或日曆，唔再留喺收集箱。
:::

## 掛到項目同掛到里程碑有咩分別

掛到項目，表示呢個任務屬於呢個項目。

掛到里程碑，表示呢個任務唔只屬於項目，仲屬於項目裏面嘅某個階段。例如一個項目裏面有「準備」「執行」「覆盤」幾個里程碑，任務可以放到其中一個里程碑下。

喺任務列表或收集箱裏面嘅「加入項目」操作通常會要求你揀到具體里程碑。咁樣做係為咗避免任務只係掛喺一個大項目下面，但唔知屬於邊個階段。

## 想將一個任務從項目裏面移出嚟

打開任務詳情，將項目欄位清空就得。

清空後，如果呢個任務亦冇截止日期，而且仲係待辦或進行中，佢會重新出現喺收集箱。如果佢仲有截止日期，佢仍然會按日期出現喺今日視圖或日曆視圖裏面。

## 一個任務可以掛多個項目嗎

唔可以。每個任務只能屬於一個項目（同一個里程碑）。

如果你發現一個任務好似跨咗兩個項目，通常有兩種處理方式：

1. 揀佢主要歸屬嗰個項目
2. 將佢拆成兩個任務，分別掛到各自嘅項目
