添加任務的規則只有一條：**寫個標題，儲存**。其他字段都是可選的，等你需要的時候再填。

## 從哪裏創建任務

| 入口 | 適合的場景 |
| --- | --- |
| 底部 **+** 按鈕 | 隨時隨地的快速錄入 |
| 收集箱頁面內的輸入框 | 專門整理收集箱時 |
| 項目或里程碑頁面內 | 直接歸屬到某個項目 |
| 已有任務詳情裏的節點 | 拆分一個大任務的步驟 |

## 任務編輯界面

![創建與編輯任務對話框](../../../screenshots/zh-hk/tasks-create-edit-dialog.png)

各字段說明：

| 字段 | 是否必填 | 作用 |
| --- | --- | --- |
| 標題 | ✅ 必填 | 任務的名稱，越具體越容易執行 |
| 描述 | 可選 | 背景信息、鏈接、備注 |
| 截止日期 | 可選 | 設置後任務出現在對應日期的任務列表 |
| 提醒 | 可選 | 在指定時間發通知，不能設在過去 |
| 項目 | 可選 | 設置後任務從收集箱移入該項目 |
| 里程碑 | 可選 | 設置後歸屬到項目的某個階段 |
| 標籤 | 可選 | 方便篩選，可以有多個 |
| 節點 | 可選 | 把任務拆成更小的步驟 |

:::tip[善用自然語言輸入]
在標題輸入框裏，你可以直接寫 `#標籤名`、`@日期`、`~提醒時間`，GranoFlow 會自動解析。詳細的解析規則見[用自然語言寫任務](title-parser)。
:::

## 儲存後任務去哪了

這取決於你填了甚麼：

- **沒有日期、沒有項目** → 進收集箱
- **有日期** → 出現在那一天的任務列表
- **有項目** → 出現在項目裏
- **在項目頁面裏創建** → 直接歸屬到那個項目

修改字段不是新建一個任務，而是改變同一個任務的歸屬。

## 編輯已有任務

點擊任何任務即可打開詳情頁。修改你想改的字段，退出時自動儲存。

:::caution[注意]
提醒不能設置在已經過去的時間。如果你設的提醒時間已經過了，系統會提示你重新選。
:::

完成、歸檔和刪除是三種不同操作，填寫字段不會讓任務自動變成完成狀態。
