title: "收集箱"
description: "收集箱是你的「先記再說」區域。先把任務記下來，之後再決定日期、項目或是否刪除。"
---

收集箱用來暫時放低那些你剛想到、但未整理好的任務：先點 **+** 記下來，之後有時間再加日期、放入項目，或者刪除不需要的事。

你可以把收集箱當成一個「先記再說」的地方。它不是最終任務列表，也不是項目管理區。它只是幫你先把事情從腦裏放出來，避免忘記。

## 怎樣把任務放進收集箱

點底部導覽列中間的 **+**，寫下任務，然後儲存。只要你沒有選日期、項目或里程碑，它就會進入收集箱。

一開始不需要想清楚所有細節。可以先這樣記：

![收集箱界面](../../../screenshots/zh-hk/tasks-inbox-main.png)

- 「明天聯絡客戶」
- 「整理課程筆記」
- 「買牙線」
- 「週末約朋友食飯」

第一步只需要把任務記下來。日期、項目和其他細節，可以之後再補。

## 為甚麼有些任務沒有出現在收集箱

收集箱不會顯示所有任務。它只顯示**沒有日期、沒有項目、沒有里程碑**，而且狀態是待辦或進行中的任務。

| 你當時做了甚麼 | 任務去了哪裏 |
| --- | --- |
| 沒選任何東西，直接儲存 | 進收集箱 ✅ |
| 選了一個日期 | 出現在那一天的任務列表 |
| 加入了某個項目 | 出現在項目裏 |
| 在項目頁面裏建立 | 自動歸屬到那個項目 |

所以，任務沒有出現在收集箱，不代表它不見了。通常只是因為它已經有日期或項目。

## 為甚麼任務會從收集箱消失

只要任務被整理過，它就會離開收集箱。例如你給它加了日期、加入了項目、完成了、封存了，或者刪除了。

收集箱不是資料夾。任務有了明確歸屬之後，就不會繼續留在這裏。

:::note[沒看到的任務在哪裏？]
去「任務列表」（有日期的任務）或具體的項目頁面裏找。整理後的任務會出現在它的新位置。
:::

## 怎樣整理收集箱

你不需要每天清空收集箱，但重要的事不要放太久。每天開始時掃一眼，按以下問題處理：

1. 這件事需要在某一天完成嗎？→ 給它加日期
2. 它屬於某個項目嗎？→ 移入對應項目
3. 它還值得做嗎？→ 不值得就刪除
4. 它現在很快就可以做完嗎？→ 三分鐘內能完成的事，可以直接做掉

:::tip[一個好習慣]
每天早上花 2 分鐘看一遍收集箱。把重要任務加上日期或項目，剩下暫時不急的可以先留着，不必強迫自己全部清空。
:::

這樣使用，收集箱就是一個幫你快速記錄、之後再整理的入口，而不是一堆令人焦慮的待處理事項。
