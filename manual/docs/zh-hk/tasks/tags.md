項目是縱向結構（任務屬於哪個目標），標籤是橫向分類（這個任務屬於哪種類型）。

## 怎麼給任務加標籤

在新建任務或打開任務詳情時，找到標籤區域，點擊即可選擇或創建標籤。

![標籤管理界面](../../../screenshots/zh-hk/tasks-tags-management.png)

- 已有標籤會作為候選直接顯示
- 找不到合適的？直接輸入新名稱創建一個
- 一個任務可以有多個標籤

## 標籤用來記甚麼好

標籤適合表達那些**跨項目、可複用**的分類：

| 用途 | 標籤示例 |
| --- | --- |
| 場景 / 精力 | `低精力` `深度工作` `碎片時間` |
| 等待狀態 | `等待他人` `等待回覆` `待確認` |
| 類型 | `電話` `創意` `管理` |
| 臨時標記 | `本週重點` `稍後處理` |

## 刪除標籤的影響

刪除一個標籤**不會**刪除使用它的任務。它只會從那些任務上解綁這個標籤。

:::caution[刪前確認]
刪除標籤是不可撤銷的。確認不再需要這個分類，再操作。
:::
