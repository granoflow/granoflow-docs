不想每次都在字段裏一個個點選？試試在標題裏直接寫：

- `#工作` — 添加一個叫「工作」的標籤
- `@明天` — 設置截止日期為明天
- `~下午3點` — 設置提醒時間

GranoFlow 會把這些識別出來，顯示成候選項，等你確認。

## 怎麼操作

輸入標題時，只要寫了 `#` `@` `~` 或日期詞，輸入框會高亮顯示識別結果。

![自然語言解析確認界面](../../../screenshots/zh-hk/tasks-title-parser-confirmation.png)

確認候選項的方式：

- 點擊高亮區域的候選項
- 按 **Enter** 或 **Tab** 確認當前候選

:::caution[重要：確認前不會寫入]
識別到的內容**不會**自動變成字段。你必須明確確認，才會寫入標籤、日期或提醒。沒有確認的部分會繼續留在標題文字裏，不會偷偷消失。
:::

## 常用語法速查

| 你輸入的 | 效果 |
| --- | --- |
| `#工作` | 候選添加「工作」標籤 |
| `@明天` `@下週五` `@3月5日` | 候選設置截止日期 |
| `~下午3點` `~明早9點` | 候選設置提醒時間 |
| `寫報告 @明天 #工作` | 同時候選日期和標籤 |

## 有甚麼限制

- `#` `@` `~` 需要在清楚的詞邊界前使用
- 解析結果只是建議，最終以你確認的字段為準
- 複雜的句子裏夾雜的符號不一定能被正確識別

如果哪次識別結果不對，直接在字段裏手動設置就好。
