## 四個詞的區別

| 詞 | 是什麼 |
|----|-------|
| **帳號** | 你在 GranoFlow 的身份 |
| **訂閱** | 你在 Apple 或 Google 平台的購買關係 |
| **會員** | GranoFlow 對外描述的用戶身份（如 Pro） |
| **權益** | 當前帳號實際能使用哪些功能 |

「我點過購買按鈕」不等於「當前帳號一定有權益」——中間每一步都可能出現狀態不對齊。

## 遇到問題時先問自己

1. 我現在登入的是哪個 GranoFlow 帳號？
2. 購買發生在哪個平台（Apple / Google）？
3. 當前訂閱是否還有效？
4. App 是否已經刷新了權益？
5. 有沒有搞混不同帳號？
