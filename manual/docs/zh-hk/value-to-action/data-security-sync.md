## 本地優先

本地優先 = 數據先存在你的設備上，不需要聯網才能用。開啟雲同步後，才會上傳加密副本到雲端。

## 端到端加密

數據離開設備前先加密。雲端存的是密文，GranoFlow 的服務器讀不到你的任務內容。

## 密鑰不是登入密碼

| | 用途 |
|--|------|
| **登入密碼 / 郵箱驗證碼** | 證明帳號屬於你 |
| **加密密鑰** | 解開你的加密數據 |

你可以順利登入帳號，但沒有正確密鑰，雲端或備份裏的數據仍然解不開。

**密鑰保存建議：** 存進可靠的密碼管理器，換設備前確認密鑰可用。

## 換設備前要做什麼

確認：舊設備能正常打開 GranoFlow → 雲同步正常或已導出備份 → 知道當前密鑰。

:::caution[空設備不是數據來源]
新設備第一次同步時，不要把空設備當成「新起點」——這會覆蓋掉你的舊數據。
:::
