---

GranoFlow 不只是一個 Todo 清單，更像一本「帶結構的生活手冊」。

```mermaid
graph TD
  A[領域] --> B[價值觀]
  B --> C[項目]
  C --> D[里程碑]
  D --> E[任務]
  E --> F[回顧]
  F --> A
```

## 領域

領域是你長期在意的生活方向，比如「工作學習」「身心健康」「業餘創作」。它不是任務分類夾，而是你人生地圖上的幾塊區域。

## 價值觀

價值觀不是目標——目標可以完成，價值觀不能被一次性打勾。

> 「三個月減重 5 公斤」→ 目標
>
> 「我希望長期照顧身體」→ 價值觀

## 項目與里程碑

項目是持續幾天到幾個月的目標容器。里程碑把大項目拆成可見的階段，讓你知道現在在推進哪一段。

## 任務

任務是今天能動手的一步。如果一個任務讓你遲遲無法開始，通常是它還不夠具體——繼續拆小。

## 回顧

回顧把完成的事連回長期方向，讓它不只是一個劃掉的清單項。

:::tip[不必一開始就搭好全部結構]
先寫任務 → 發現它會持續就建項目 → 項目變大再拆里程碑。結構慢慢長出來的。
:::
