---
title: "帳號刪除與資料處理"
description: "刪除帳號會清除雲端資料，但不會自動清除每台裝置上的本地副本——操作前先搞清楚影響範圍。"
---

如果你想知道「刪除帳號後資料會怎樣」，答案是：雲端資料會在寬限期後永久刪除，訂閱權益會失效；但每台裝置裡的本地 GranoFlow 資料不會自動消失，需要你自己處理。

## 刪帳號會發生什麼事

| 項目 | 刪帳號後 |
| --- | --- |
| 雲端資料 | 在寬限期後永久刪除 |
| 各裝置本地資料 | **不會自動刪除** |
| 訂閱權益 | 帳號刪除後權益失效 |
| 可以撤銷嗎 | 寬限期內可以取消，之後不可逆 |

## 刪帳號前的檢查清單

- [ ] 確認你不再需要雲端同步或多裝置共享
- [ ] 如果你還想保留資料，先匯出本地備份
- [ ] 如果你有進行中的訂閱，先了解退款政策
- [ ] 如果你有多台裝置，先確認每台裝置上還有哪些本地資料

## 刪帳號 vs 登出

- **登出**：只會中斷這台裝置和帳號的連線。本地資料還在，之後可以重新登入。
- **刪除帳號**：註銷帳號。雲端資料會在寬限期後刪除，寬限期結束後無法恢復。

如果你只是暫時不想用 GranoFlow，通常登出就夠了。刪除帳號適合你確定不再需要這個帳號和雲端資料的情況。

## 寬限期

提交帳號刪除申請後，通常會有一段寬限期，實際時長請看 App 內說明。在寬限期內，你可以取消刪除申請，讓帳號恢復正常。寬限期結束後，刪除操作不可撤銷。

:::caution[本地資料不會自動消失]
刪除帳號後，你每台裝置上的本地 GranoFlow 資料仍然存在，直到你手動解除安裝 App 或清除 App 資料。如果你不想保留這些本地資料，需要在對應裝置上手動清除。
:::
