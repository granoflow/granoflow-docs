如果你擔心圖片是不是會跟任務一起安全保存，先記住一句話：文字任務和圖片附件是分開儲存、分開同步的。文字可能已經出現在另一台裝置，圖片還在上傳；「清空本地附件（保留圖片）」只會清掉這台裝置上的非圖片附件，不會透過這個入口清掉圖片。

![附件管理入口](../../../screenshots/zh-tw/subscription-vip-settings.png)

## 圖片和文字有什麼不同

- **文字任務**：內容小，同步快，通常很快就能在本機和雲端保持一致
- **圖片附件**：檔案大，同步慢，可能在文字任務已經同步後還沒傳完

所以，你在另一台裝置看到任務已經出現，但圖片暫時沒有顯示，不一定代表任務遺失。通常先確認網路穩定，再給圖片一些上傳和下載時間。

## 刪除圖片附件是什麼意思

<!-- manual-screenshot:id=data-attachments-clear-detail -->
![附件與圖片 介面截圖](../../../screenshots/zh-tw/data-attachments-clear-detail.png)

處理圖片相關內容時，要分清楚兩件事：

- **從任務移除**：只是不再讓這個任務關聯這張圖片；這台裝置上的本機檔案可能還在
- **清空本地附件（保留圖片）**：從 `設定 → Pro 設定` 進入，在未開啟全量同步附件時清空這台裝置上的非圖片附件，用來釋放本機儲存空間；開啟全量同步附件時，這個按鈕會停用

清空本地附件以後，被清掉的非圖片附件之後通常可以重新下載。圖片是否能在其他裝置看到，仍取決於圖片是否已經成功上傳到雲端；如果圖片從來沒有成功上傳，換裝置或清除 App 本機資料後就沒有可恢復的來源。

## 本機備份會包含圖片嗎

本機備份會打包目前裝置上已有的業務記錄和附件檔案，包括已經落在這台裝置上的圖片、PDF 和通用附件。但它不會幫你從雲端補齊這台裝置尚未下載的附件；如果圖片或附件還沒有上傳成功，或者雲端附件還沒有落到本機，備份裡也可能缺少它們。如果你需要附件盡量完整，關鍵是讓雲端同步維持正常狀態；Pro 用戶還應先開啟「全量同步附件」，等同步完成後再建立備份。

:::note[圖片上傳需要網路]
圖片不會離線上傳。例如你在捷運裡拍了一張照片並附到任務上，這張圖片要等到下次有網路時，才會繼續上傳到雲端。
:::
