你現在最需要做的是：在還能打開 GranoFlow 的裝置上，把關鍵金鑰儲存到密碼管理器或安全的地方。以後換裝置、重裝 App、匯入本機備份，或看到「雲端同步還原」時，GranoFlow 可能會要求你輸入對應金鑰；沒有它，某些加密資料可能打不開。

GranoFlow 的備份和雲端同步都使用加密。金鑰像保險箱鑰匙：**沒有正確的鑰匙，連 GranoFlow 自己的伺服器也無法讀取被保護的資料**。這也意味著：**如果你自己把金鑰弄丟了，GranoFlow 無法幫你重設或找回已經被它保護的資料。**

這裡有兩個用戶會看到的名字：建立本機備份時顯示的是「資料密鑰」；雲端同步還原裡輸入的是「雲端同步密碼」。它們底層都服務於加密保護，但出現的場景不同。不要把它們當作登入密碼，也不要把它們只存在 GranoFlow 裡。

## 金鑰 vs 登入密碼，有什麼區別

<!-- markdownlint-disable MD060 -->
|  | 登入密碼（或驗證郵件） | 資料密鑰 / 雲端同步密碼 |
| --- | --- | --- |
| 用來做什麼 | 證明你是誰 | 打開雲端加密資料 |
| 忘了怎麼辦 | 重新發驗證郵件 | **無法找回** |
| 換了會怎樣 | 只影響登入 | 影響是否能存取雲端資料 |
<!-- markdownlint-enable MD060 -->

## 資料密鑰在哪裡出現

建立本機備份時，GranoFlow 會在備份開始前顯示「資料密鑰」。它預設隱藏，點眼睛圖示才顯示。你需要先把它抄錄並妥善保存，再勾選確認，才能開始建立備份。

這把資料密鑰對應目前和之後用它建立的備份。重新產生新資料密鑰，不會讓舊備份改用新密鑰。也就是說，舊備份仍然需要舊密鑰。儲存備份檔案時，請同時儲存當時的資料密鑰。

## 雲端同步密碼在哪裡出現

當新裝置加入既有雲端同步，或當前裝置和雲端同步資料的密鑰不一致時，GranoFlow 會進入「雲端同步還原」，並要求輸入「雲端同步密碼」。它用於嘗試打開雲端已有的加密同步資料。

輸入正確後，GranoFlow 還會繼續判斷本機資料和雲端資料是否屬於同一套資料：

- 同一套資料：只更新這台裝置的同步保護設定。
- 不是同一套資料：進入使用雲端資料或保留本機路徑的選擇流程。
- 輸入不正確：不儲存輸入，不下載、不上傳、不清空。

## 應該怎麼保存

建議把金鑰儲存到你離開當前裝置後仍能存取的地方，例如密碼管理器、安全筆記、離線紙本記錄或加密儲存。不要只儲存截圖，也不要只存在 GranoFlow 裡，因為需要金鑰時，往往正是你換了裝置、重裝系統，或舊環境打不開的時候。

一個簡單做法是給每條金鑰寫清楚用途：

- `GranoFlow 本機備份 2026-06-18 資料密鑰`
- `GranoFlow 雲端同步密碼`
- `舊電腦遷移前備份`

這樣以後匯入備份或還原同步時，你能快速判斷該用哪一把。

## 在新裝置上什麼時候需要金鑰

以下情況可能需要你輸入舊裝置或舊備份對應的金鑰：

- 換了手機或電腦
- 重裝了 GranoFlow
- 匯入 `.flow.grano` 本機備份
- 看到「雲端同步還原」或雲端同步密碼輸入提示

輸入正確金鑰後，新裝置才有機會存取對應的備份或雲端加密資料。

## 輸入金鑰後會發生什麼

GranoFlow 會先檢查這把金鑰能不能打開目標資料。匯入備份時，目標是備份檔案；雲端同步還原時，目標是當前帳號的雲端同步資料。

- **金鑰匹配，且資料屬於同一套** → 繼續匯入或連接同步。
- **金鑰匹配，但本機和雲端不是同一套資料** → 顯示選擇介面，讓你決定保留哪份資料。
- **金鑰不對** → 不改變任何資料，讓你重新輸入。

## 忘了金鑰怎麼辦

按這個順序檢查：

1. **舊裝置還能用嗎？** → 在舊裝置上打開 GranoFlow，找到金鑰並複製
2. **密碼管理器裡有嗎？** → 檢查你常用的密碼管理器
3. **備份檔案旁邊有備註嗎？** → 檢查備份目錄、雲端硬碟備註、安全筆記或遷移清單
4. **舊裝置還在，但 App 打不開？** → 聯絡 GranoFlow 支援，說明舊裝置和當前情況

如果以上都沒有，雲端加密資料可能無法還原。本機備份（如果有的話）仍然可用。

## 看到「輸入雲端同步密碼」時怎麼辦

如果 GranoFlow 提示你輸入雲端同步密碼，請填入當前雲端資料對應的完整金鑰。不要先清空雲端，也不要隨便輸入新產生的本機資料密鑰來試。

輸入正確後，GranoFlow 會先判斷本機資料和雲端資料是不是同一份：

- 如果是同一份資料，只會更新這台裝置的同步金鑰設定。
- 如果不是同一份資料，才會進入「使用雲端資料」的確認流程。繼續前請先確認本機資料和雲端資料哪一份更重要。

:::caution[金鑰不是密碼，不能重設]
資料密鑰和雲端同步密碼遺失後，GranoFlow 無法幫你重設或找回已被它們保護的資料。現在就儲存好，不要等到換裝置或匯入備份時才發現缺了一把鑰匙。
:::
