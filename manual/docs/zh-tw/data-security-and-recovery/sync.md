多端同步的意思是：你在一台裝置新增、修改或刪除內容後，其他登入同一個帳號的裝置，也會盡量變成相同狀態。它方便你在多台裝置繼續使用 GranoFlow，但它不是用來救回誤刪內容的備份功能。

![同步狀態介面](../../../screenshots/zh-tw/data-sync-status-main.png)

## 同步什麼、不同步什麼

✅ 會同步的內容：

- 任務，例如標題、截止日期、標籤、狀態等
- 專案和里程碑
- 回顧紀錄
- 圖片和附件，在網路允許的情況下同步

⚠️ 最重要的是：同步不是備份。

- **你刪了，其他裝置也會刪**：同步是雙向的，刪除操作不會只留在一台裝置上。
- **沒有版本歷史**：同步不會保存「3 天前長什麼樣子」。
- **圖片可能晚一點出現**：文字內容可能先同步完成，圖片和附件可能稍後才同步完成。

## 同步的常見狀態

| 狀態 | 含義 |
| --- | --- |
| 同步中 | 正在上傳或下載變更。 |
| 已同步 | 目前裝置的資料和雲端一致。 |
| 等待中 | 有變更正在排隊，通常和網路有關。 |
| 錯誤 | 同步遇到問題，需要檢查帳號或金鑰。 |

## 離線和服務無法連線時會怎樣

如果暫時沒有網路，GranoFlow 裡的本機資料仍然可以使用。你可以繼續記錄任務、整理專案、寫回顧、搜尋內容，也可以匯出本機備份。

如果你開啟了「離線模式」，同步、登入、訂閱確認和恢復購買會暫時無法使用。你點選這些需要連線的功能時，App 會提示目前無法連接對應服務；關閉離線模式後，可以再試一次。

如果裝置有網路，但同步服務暫時無法連線，App 也不會阻止你使用本機已有的資料。已經在這台裝置上的內容仍然可以編輯；需要上傳到雲端或從雲端下載的變更，會等你稍後再次同步。

會員裝置可以在「設定 > 訊息與提醒」中開啟靜默同步。靜默同步用於讓其他裝置的新資料自動補到本機，但它保持靜默：不會彈系統橫幅，不會播放聲音，也不會新增通知中心訊息。是否能開啟仍取決於登入狀態、會員權益和目前平台是否支援。

## 新設備加入同步

如果你換了新手機，或重新安裝 App，想接上原本的雲端資料，需要使用舊裝置上的**雲端同步金鑰**。

詳細步驟見 → [新設備同步已有雲端資料](/manual/zh-tw/data-security-and-recovery/new-device-sync/)

:::caution[同步不等於備份]
請定期匯出本機備份。誤刪的任務不能靠同步復原，因為雲端和其他裝置也會一起刪除。
:::
