---
title: "桌面端安裝"
description: "下載、安裝 GranoFlow 桌面端，以及首次開啟時遇到系統攔截怎麼處理。"
---

如果你想在電腦上使用 GranoFlow，請先到 GranoFlow 官網下載安裝檔，安裝後用同一個帳號登入。桌面端可以留在電腦上使用，也可以用快捷鍵叫出，不需要每次都先開瀏覽器。

## 下載安裝

打開 [GranoFlow 官網](https://granoflow.com)，下載適合你電腦系統的安裝檔：

- **macOS**：下載 `.dmg` 檔案，開啟後把 GranoFlow 拖到「應用程式」資料夾。
- **Windows**：下載 `.exe` 安裝包，然後按照安裝精靈一步步完成安裝。

## macOS 提示「無法開啟」怎麼處理

第一次開啟 GranoFlow 時，macOS 可能會提示它來自不明開發者，導致你不能直接開啟。

可以這樣處理：

1. 打開 Finder，找到 GranoFlow 應用程式。
2. 右鍵點選 GranoFlow。
3. 選擇「開啟」。
4. 系統再次跳出確認視窗時，再點選「開啟」。

也可以從系統設定裡處理：打開「系統設定」→「隱私權與安全性」，找到 GranoFlow 的攔截記錄，然後點選「仍然開啟」。

## Windows 提示 SmartScreen 怎麼處理

第一次執行 GranoFlow 時，Windows Defender SmartScreen 可能會跳出警告。

在這個提示裡，先點選「更多資訊」，再點選「仍要執行」。

## 安裝完成後

安裝完成後，打開 GranoFlow，並用你的帳號登入。如果這個帳號之前已經有雲端資料，GranoFlow 會開始同步這些資料。

:::note[桌面端和手機端共用同一帳號]
在桌面端和手機端登入同一個帳號後，資料會透過雲端同步，在不同裝置之間共用。
:::
