---
title: "桌面端安裝"
description: "下載、安裝 GranoFlow 桌面端，以及首次開啟時遇到系統攔截怎麼處理。"
---

桌面端讓 GranoFlow 常駐在你的電腦上，隨時用快捷鍵呼叫，不需要每次開啟瀏覽器。

## 下載安裝

從 [GranoFlow 官網](https://granoflow.com) 下載對應系統的安裝包：

- **macOS**：下載 `.dmg` 檔案，開啟後拖入應用程式資料夾
- **Windows**：下載 `.exe` 安裝包，按照安裝精靈完成

## macOS 提示「無法開啟」怎麼處理

第一次開啟時，macOS 可能提示「來自不明開發者」。

解決方法：

1. 在 Finder 裡右鍵點選應用程式 → 選擇「開啟」
2. 再次點選「開啟」確認

或者：系統設定 → 隱私權與安全性 → 找到 GranoFlow 的攔截記錄 → 點選「仍然開啟」。

## Windows 提示 SmartScreen 怎麼處理

Windows Defender SmartScreen 可能彈出警告。點選「更多資訊」→「仍要執行」即可。

## 安裝完成後

首次開啟後用你的帳號登入，GranoFlow 會自動開始同步雲端資料（如果你之前有雲端資料的話）。

:::note[桌面端和手機端共用同一帳號]
登入同一帳號，資料會透過雲端同步在桌面端和手機端之間共用。
:::
