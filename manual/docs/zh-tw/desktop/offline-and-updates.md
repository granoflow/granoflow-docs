## 離線時能做什麼

沒有網路時，GranoFlow 桌面端仍然可以：

- 查看、新增、編輯任務
- 寫日記和回顧
- 查看專案和里程碑

不能做的：

- 同步到其他設備（等連線後自動同步）
- 同步雲端新變更到本機

## 重新連線後

連線後，GranoFlow 會自動把離線期間的變更同步到雲端。如果多台設備都在離線期間做了修改，同步時可能會合併衝突。

## 版本更新

GranoFlow 桌面端會在有新版本時提示更新。

- **不更新會怎樣**：可以繼續使用目前版本，但可能無法使用新功能，或在未來某個時間點需要強制更新
- **更新會清除資料嗎**：不會。更新只替換應用程式檔案，你的本機資料和帳號設定不受影響

:::note[強制更新]
如果伺服器端做了不相容的變更，GranoFlow 可能會要求強制更新才能繼續使用。
:::
