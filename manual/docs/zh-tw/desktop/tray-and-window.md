如果你關掉 GranoFlow 桌面視窗，但應用程式好像還在執行，這通常是正常的：**關閉視窗只是隱藏視窗，不等於結束應用程式**。如果要完全關閉 GranoFlow，需要從 Dock、選單列或系統匣選擇結束。

![系統工作列與視窗](../../../screenshots/zh-tw/desktop-tray-window-behavior.png)

## 關閉視窗後會發生什麼

在 macOS 和 Windows 上，點選視窗關閉按鈕（×）時，GranoFlow 通常只是把視窗隱藏起來。

這時應用程式仍然在背景執行。也就是說，你看不到主視窗，但 GranoFlow 還沒有結束，同步也會繼續。

這樣設計是為了讓你之後能更快重新開啟視窗，也避免背景同步中斷。

## 如何再次開啟視窗

如果視窗被隱藏，可以這樣開回來：

- **macOS**：點選選單列裡的 GranoFlow 圖示
- **Windows**：點選工作列系統匣裡的 GranoFlow 圖示
- **任何平台**：如果你設定了全域快捷鍵，也可以直接按快捷鍵叫出

## 如何真正結束應用程式

如果你想讓 GranoFlow 停止在背景執行，需要手動結束應用程式：

- **macOS**：右鍵點選 Dock 或選單列裡的 GranoFlow 圖示，然後選擇「結束」
- **Windows**：右鍵點選系統匣裡的 GranoFlow 圖示，然後選擇「結束」

真正結束後，GranoFlow 不再在背景執行，同步也會暫停，直到你下次重新開啟應用程式。

:::tip[想每次關視窗就結束？]
你可以在 GranoFlow 偏好設定裡修改關閉視窗的預設行為。
:::
