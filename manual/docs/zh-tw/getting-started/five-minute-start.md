當你想用最短路徑進入 GranoFlow 這款個人規劃應用時，從這頁開始。不需要先重建整個生活系統。

## 先捕捉一件事

打開快速新增，寫下腦中最清楚的一個任務或想法。第一分鐘可以把 GranoFlow 當成 Todo 應用：先記下來，再決定它屬於哪裡。

## 重要時再連接

如果這件事屬於更長方向，把它連接到專案或里程碑。這就是一個日計劃動作如何成為目標規劃、專案規劃器和個人時間結構的一部分。

## 回顧，不給自己打分

一天結束前，看一眼完成了什麼，再寫下下一步。日回顧不是分數、連續打卡或羞辱機制，而是讓長期目標和複盤保持連接的安靜方式。
