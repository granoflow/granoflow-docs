title: "5 分鐘上手"
description: "先記一個任務，需要時連接到專案，再用一次簡短回顧結束今天。"
---

想最快開始用 GranoFlow，就先做三件事：新增一個任務，需要時把它連接到專案，今晚用一兩句話回顧今天。不需要先建立完整系統。

## 第一步：先寫下一個任務

打開 GranoFlow，點 **+**，寫下你現在腦中最清楚的一件事。

先不用判斷它重不重要，也不用急著決定它屬於哪個專案。第一步只是先記下來。新任務會先進入收集箱，之後你有時間再整理。

![快速新增任務面板](../../../screenshots/zh-tw/interface-quick-add-main.png)

:::tip[快捷鍵]
桌面端可以用全域快捷鍵直接叫出快速新增，不用先切換到 GranoFlow 視窗。快捷鍵可以在設定裡自訂。
:::

## 第二步（可選）：需要時連接到專案

如果這件事屬於一個更大的方向，例如「完成論文」或「準備搬家」，你可以在任務詳情裡把它連接到對應專案。

這一步不是必要的。你可以先把任務留在收集箱，等專案結構真的有幫助時再整理。

## 第三步：今晚做一次小回顧

今天結束前，打開 GranoFlow，看一眼今天完成了什麼、還有什麼沒完成，然後在日回顧寫一兩句：

- 今天做了什麼
- 明天最想推進哪一件事

不用寫長篇總結，不用打分，也不用變成每天必做的壓力。它只是幫你把今天、明天和長期目標輕輕連起來。
