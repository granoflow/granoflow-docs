如果你想管理一個較大的目標，比如一次發布、一門課程或一段時間的計畫，就可以建立專案：進入專案頁，點擊右上角 **+**，填寫專案名稱和必要說明，然後建立。

## 建立入口

在專案列表頁右上角點擊 **+** 按鈕，會跳出建立對話框。

![專案列表入口](../../../screenshots/zh-tw/projects-overview-main.png)

建立時可以填寫：

- **名稱**（必填）：寫清楚這個專案要完成什麼。比如「Q3 產品發布」比「專案一」更容易看懂。
- **描述**（可選）：記錄背景、範圍、限制或交付說明。之後打開專案時，它會作為專案上下文保留下來。
- **領域**（可選）：把專案放到一個大方向下，比如「工作」「學習」「健康」。

:::tip[名字起好，以後省事]
專案名稱會出現在專案列表裡。建議用「動詞 + 目標」的寫法，比如「寫完畢業論文」，不要只寫「論文」。
:::

## 建立空專案 vs 先建專案再加任務

兩種方式都可以，按你的使用習慣選就行：

| 方式 | 適合場景 |
| --- | --- |
| 先建立專案，再慢慢加任務 | 還在規劃階段，想先把專案架構建起來 |
| 先把任務加到收集箱，再歸入專案 | 正在執行，想先快速記下來，之後再整理 |

不管用哪種方式，最後任務都可以放進專案裡管理。

## 專案建好之後

專案建立後，你通常可以繼續做這些事：

1. 新增里程碑（可選）——如果這個目標有明顯階段。
2. 把現有任務連接到專案和某個里程碑。
3. 在里程碑下直接建立新任務。

如果你現在還不知道要加哪些任務，也可以先關閉專案頁面。專案已經建立好，之後再回來補充就行。
