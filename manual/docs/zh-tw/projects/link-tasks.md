把任務連結到專案後，它會離開收件匣，並出現在專案裡；如果任務有日期，也仍然會出現在今日視圖或日曆視圖。專案用來說明這件事屬於哪個專案，里程碑用來說明它屬於專案裡的哪個階段。

## 怎麼連結

你可以用兩種方式把任務連結到專案。

### 方法一：在任務詳情裡選擇專案

打開任務詳情，找到專案欄位，選擇目標專案。需要時，也可以繼續選擇這個專案下的某個里程碑。

這是最常用的方式，適合你已經打開任務、想直接幫它分類的時候。

### 方法二：在專案頁面拖入任務

打開專案詳情頁，把現有任務拖曳到某個里程碑下面。這樣任務就會連結到這個專案，並歸到對應里程碑下。

## 任務連結後出現在哪裡

連結專案之後，同一個任務可能會同時出現在多個地方。它不是被複製了，而是同一個任務在不同視圖裡顯示。

| 視圖 | 說明 |
| --- | --- |
| 專案頁面 | 出現在對應專案裡；如果選了里程碑，會出現在對應里程碑下 |
| 今日視圖 | 如果任務有今天的日期，仍然出現 |
| 日曆視圖 | 依截止日期顯示 |
| 收件匣 | **不再出現**（掛上專案後會離開收件匣） |

:::note[收件匣的變化]
只有同時滿足「無截止日期」「無專案」「無里程碑」的任務，才會待在收件匣。一旦你把任務掛進專案，它就會自動離開收件匣，這是正常行為。
:::

## 掛到專案和掛到里程碑有什麼差別

掛到專案，表示這個任務屬於這個專案。

掛到里程碑，表示這個任務不只屬於專案，還屬於專案裡的某個階段。例如一個專案裡有「準備」「執行」「回顧」幾個里程碑，任務可以放到其中一個里程碑下。

如果你還沒想好任務屬於哪個階段，只掛到專案也可以。

## 想把一個任務從專案裡移出來

打開任務詳情，把專案欄位清空就可以。

清空後，如果這個任務也沒有截止日期，它會重新出現在收件匣。如果它還有截止日期，它仍然會依日期出現在今日視圖或日曆視圖。

## 一個任務可以掛多個專案嗎

不可以。每個任務只能屬於一個專案（和一個里程碑）。

如果你發現一個任務好像橫跨兩個專案，通常有兩種處理方式：

1. 選擇它更主要歸屬的那個專案
2. 把它拆成兩個任務，分別掛到各自的專案
