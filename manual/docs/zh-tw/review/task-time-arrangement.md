title: "自動擺布任務時間"
description: "當日回顧裡多個任務時間重疊時，可以把當天任務重新排成一條比較容易檢查的時間線。"
---

如果你經常忙完之後才補任務，日回顧裡可能會出現幾個任務擠在同一個時間點。自動擺布任務時間，就是先把這些記錄放進一條更接近當天節奏的預覽時間線，方便你回看、拖動微調，再決定是否儲存。

這不是行事曆排班，也不是證明真實發生時間的工具。任務時間在 GranoFlow 裡仍然可以重疊；這個入口只是給事後回顧一個比較輕的整理入口。

## 從哪裡開始

打開日回顧。如果右側欄提示任務時間有重疊，你會看到「自動擺布任務時間」入口。
沒有重疊任務時，這條提示不會顯示；你仍然可以直接使用「回顧今日任務」。

## 自動擺布會怎麼做

第一版預設依照輕量規則處理：

| 規則 | 含義 |
| --- | --- |
| 開始時間 | 當天 9:00 |
| 結束時間 | 你點擊排布時的目前時間 |
| 排布方式 | 依照任務順序平均分配 |
| 時間線 | 在預覽彈窗內可以重疊，任務之間也可以留空隙 |
| 底層規則 | 不改變任務時間可以重疊的合法性 |

任務順序會優先參考完成順序、建立順序和目前顯示順序。點擊入口後不會立刻改任務時間；GranoFlow 會先打開底部預覽彈窗，你可以檢查結果是否合理，再儲存。
## 排布後怎麼檢查

預覽彈窗旁邊也有時間軸。你可以拖動任務區塊上沿改開始時間，拖動下沿改結束時間；任務區塊不能橫向拖動。桌面端也可以直接在任務區塊輸入開始和結束時間，再用 Tab 鍵切換到下一個輸入框。

重點看三件事：

1. 總時間區間是否接近你今天實際投入的範圍。
2. 任務順序是否大致符合你當天做事的先後。
3. 單個任務的時間是否明顯過長或過短。

如果只是為了讓日回顧更容易看，不需要把每分鐘都調到絕對準確。你只要把明顯擠在一起的記錄整理開，就已經足夠有用。

預覽彈窗內允許任務時間重疊，方便你把某一項直接拖到或輸入到更接近實際的時間段；儲存前 GranoFlow 仍會檢查開始時間早於結束時間，並且開始時間不能早於任務建立時間。

## 和回顧今日任務的關係

「自動擺布任務時間」先幫你整理時間區塊。「回顧今日任務」則讓 AI 和你一起檢查每個任務的耗時是否合理，並在確認後寫回開始時間、結束時間、標題或任務回顧。

如果你不確定怎麼分配時間，可以先自動擺布，再使用[回顧今日任務](../ai-assistance/daily-task-review)繼續細化。
