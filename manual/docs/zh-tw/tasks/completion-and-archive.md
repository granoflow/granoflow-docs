任務從清單裡不見了，不代表它不見了。GranoFlow 有三個地方可能「藏著」你的任務：

- **已完成**：做完了，去了完成視圖
- **已封存**：暫時不需要，但留著記錄
- **回收桶**：刪了，還沒清空

## 完成

做完一件事後，把它標記為完成。它會從待辦清單消失，記錄完成時間，並出現在「已完成」視圖和日回顧統計裡。

![已完成任務和封存介面](../../../screenshots/zh-tw/tasks-completed-archived-trash.png)

:::tip[小技巧]
已完成的任務不要隨手刪掉——它們是你的日回顧資料。
:::

## 封存

封存適合那些「這件事不需要每天盯著，但要留著記錄」的任務。

![已封存任務清單](../../../screenshots/zh-tw/tasks-archived-list.png)

封存和完成的區別：
- **完成**：表示任務真的做完了，會進入統計
- **封存**：只是從當前視圖收起，不代表做完，也不進入完成統計

## 回收桶

刪除任務後，任務進入回收桶，還有機會還原。

![回收桶清單](../../../screenshots/zh-tw/tasks-trash-list.png)

:::caution[清空前想好]
手動清空回收桶是不可復原的。
:::

## 找不到任務怎麼辦

按順序檢查：篩選條件 → 日期清單 → 專案頁面 → 已完成 → 已封存 → 回收桶
