有什麼事冒出來了？先記下來，別想那麼多。

收集箱（Inbox）就是幹這個用的——它是個臨時停靠區，專門放那些「你還不知道放哪裡」的任務。等有空了，再一件件整理好。

把它想成一個永遠不會滿的收件匣。不是你的工作場所，只是你的緩衝帶。

## 怎麼把任務放進收集箱

點底部導覽列中間的 **+**，寫下任務，儲存，就進收集箱了。

不需要選日期，不需要選專案，不需要加標籤。越簡單越好：

![收集箱介面](../../../screenshots/zh-tw/tasks-inbox-main.png)

- 「明天聯繫客戶」
- 「整理課程筆記」
- 「買牙線」
- 「週末約朋友吃飯」

第一步只是把它從腦子裡「卸載」出來。細節等你有空再填。

## 為什麼有些任務沒出現在收集箱

收集箱不是放所有任務的地方，它只顯示**沒有日期、沒有專案、沒有里程碑**的待辦或進行中任務。

| 你當時做了什麼 | 任務去哪了 |
| --- | --- |
| 沒選任何東西，直接儲存 | 進收集箱 ✅ |
| 選了一個日期 | 出現在那一天的任務清單 |
| 加入了某個專案 | 出現在專案裡 |
| 在專案頁面裡建立 | 自動歸屬到那個專案 |

任務沒在收集箱不代表它不見了，只是它已經有了歸屬。

## 為什麼任務會從收集箱裡消失

給任務加了日期、加入了專案、完成了、封存了、刪除了——只要滿足其中一個，任務就會自動離開收集箱。

收集箱不是資料夾，整理好的任務不會留在這裡。

:::note[沒看到的任務在哪？]
去「任務清單」（有日期的）或具體的專案頁面裡找——那裡才是它整理後的新家。
:::

## 怎麼整理收集箱

不需要每天清空，但別讓重要的事在這裡放太久。每天開始時掃一眼，問自己：

1. 這件事需要在某天完成嗎？→ 給它加個日期
2. 它屬於某個專案嗎？→ 移進對應專案
3. 它還值得做嗎？→ 不值得就刪掉
4. 它現在就可以做完嗎？→ 三分鐘內的事，直接做

:::tip[一個好習慣]
每天早上花 2 分鐘掃一遍收集箱，把真正重要的任務給出日期或專案。剩下的先放著，不用強迫自己全清空。
:::

這樣你的收集箱就是一個「先記後整」的工具，而不是一個讓你焦慮的待處理堆。
