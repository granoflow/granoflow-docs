title: "收集箱"
description: "收集箱是你的「先記再說」區域。先把任務記下來，之後再決定日期、專案或是否刪除。"
---

收集箱用來暫時放那些你剛想到、還沒整理好的任務：先點 **+** 記下來，等有空再替它加日期、放進專案，或刪掉不需要的事。

你可以把收集箱當成一個「先記再說」的地方。它不是最終的任務清單，也不是專案管理區。它只是幫你先把事情從腦中放下來，避免忘記。

## 怎麼把任務放進收集箱

點底部導覽列中間的 **+**，寫下任務，然後儲存。只要你沒有選日期、專案或里程碑，它就會進入收集箱。

一開始不需要想清楚所有細節。可以先這樣記：

<!-- manual-screenshot:id=tasks-inbox-main -->
![收集箱 介面截圖](../../../screenshots/zh-tw/tasks-inbox-main.png)

- 「明天聯絡客戶」
- 「整理課程筆記」
- 「買牙線」
- 「週末約朋友吃飯」

第一步只要把任務記下來。日期、專案和其他細節，可以之後再補。

## 為什麼有些任務沒出現在收集箱

收集箱不會顯示所有任務。它只顯示**沒有日期、沒有專案、沒有里程碑**，而且狀態是待辦或進行中的任務。

| 你當時做了什麼 | 任務去哪了 |
| --- | --- |
| 沒選任何東西，直接儲存 | 進收集箱 ✅ |
| 選了一個日期 | 出現在那一天的任務清單 |
| 加入了某個專案 | 出現在專案裡 |
| 在專案頁面裡建立 | 自動歸屬到那個專案 |

所以，任務沒出現在收集箱，不代表它不見了。通常只是因為它已經有日期或專案。

## 為什麼任務會從收集箱裡消失

只要任務被整理過，它就會離開收集箱。比如你替它加了日期、加入了專案、完成了、封存了，或刪除了。

收集箱不是資料夾。任務有了明確歸屬之後，就不會繼續留在這裡。

:::note[沒看到的任務在哪裡？]
去「任務清單」（有日期的任務）或具體的專案頁面裡找。整理後的任務會出現在它的新位置。
:::

## 怎麼整理收集箱

你不需要每天清空收集箱，但重要的事不要放太久。每天開始時掃一眼，照下面的問題處理：

1. 這件事需要在某天完成嗎？→ 替它加日期
2. 它屬於某個專案嗎？→ 移進對應專案
3. 它還值得做嗎？→ 不值得就刪除
4. 它現在很快就能做完嗎？→ 三分鐘內能完成的事，可以直接做掉

:::tip[一個好習慣]
每天早上花 2 分鐘看一遍收集箱。把重要任務加上日期或專案，剩下暫時不急的可以先留著，不必強迫自己全部清空。
:::

這樣使用，收集箱就是一個幫你快速記錄、之後再整理的入口，而不是一堆讓你焦慮的待處理事項。
