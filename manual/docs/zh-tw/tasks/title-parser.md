如果你想快速幫任務加標籤、截止日期、專案歸屬或提醒，可以直接寫在快速新增任務的標題裡。例如輸入 `寫報告 明天 #工作 @論文初稿 ~下午3點`，GranoFlow 會把日期、標籤、專案或里程碑，以及提醒顯示成候選項；你確認後，才會寫入對應欄位。

## 怎麼操作

快速新增任務時，直接輸入這些內容：

- `#工作`：候選新增一個叫「工作」的標籤
- `@論文初稿`：候選關聯一個專案或里程碑
- `明天`、`下週五`、`3月5日`：候選設定截止日期
- `~下午3點`：候選設定提醒時間

只要 GranoFlow 識別到 `#`、`@`、`~` 或日期詞，輸入框裡會顯示對應候選項。

標籤、專案、里程碑和提醒候選項可以這樣確認：

- 點擊高亮區域裡的候選項
- 按 **Enter** 或 **Tab** 確認目前候選項

日期候選也需要確認。你可以點擊日期候選；如果剛輸入了清楚的日期詞，也可以在日期詞後按空格，讓 GranoFlow 把它寫入截止日期欄位。

:::caution[重要：確認前不會寫入]
識別到的內容**不會**自動變成標籤、日期、專案、里程碑或提醒。你必須明確確認，才會寫入對應欄位。沒有確認的部分會繼續留在標題文字裡，不會偷偷消失。
:::

專案、里程碑和標籤候選項也遵守同一規則。候選清單只幫你選擇；只有在你點擊候選項，或用 Enter / Tab 確認後，欄位才會保存。如果你按 Escape 或沒有確認候選項，這段文字會作為普通標題文字保留。

## 常用語法速查

| 你輸入的 | 效果 |
| --- | --- |
| `#工作` | 候選新增「工作」標籤 |
| `@論文初稿` `@官網改版` | 候選關聯專案或里程碑 |
| `明天` `下週五` `3月5日` | 候選設定截止日期 |
| `~下午3點` `~明早9點` | 候選設定提醒時間 |
| `寫報告 明天 #工作 @論文初稿` | 同時候選日期、標籤和專案或里程碑 |

## 有什麼限制

- `#`、`@`、`~` 最好放在清楚的詞邊界前，例如 `#工作`，不要寫成 `搞#工作`
- `@` 只用於專案或里程碑，不用於日期。要設定日期，直接寫 `明天`、`下週五` 這類日期詞
- 解析結果只是建議，最終以你確認後的欄位為準
- 複雜句子裡夾雜的符號，不一定都能被正確識別

如果某次識別結果不對，直接在欄位裡手動設定就好。自然語言輸入只是快捷方式，不是必須使用的功能。
