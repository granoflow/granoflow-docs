如果你想快速幫任務加標籤、截止日期或提醒，可以直接寫在任務標題裡。例如輸入 `寫報告 @明天 #工作 ~下午3點`，GranoFlow 會把日期、標籤和提醒顯示成候選項；你確認後，才會寫入對應欄位。

## 怎麼操作

新增或編輯任務標題時，直接輸入這些內容：

- `#工作`：候選新增一個叫「工作」的標籤
- `@明天`：候選設定截止日期為明天
- `~下午3點`：候選設定提醒時間

只要 GranoFlow 識別到 `#`、`@`、`~` 或日期詞，輸入框裡會高亮顯示對應候選項。

你可以用兩種方式確認候選項：

- 點擊高亮區域裡的候選項
- 按 **Enter** 或 **Tab** 確認目前候選項

:::caution[重要：確認前不會寫入]
識別到的內容**不會**自動變成標籤、日期或提醒。你必須明確確認，才會寫入對應欄位。沒有確認的部分會繼續留在標題文字裡，不會偷偷消失。
:::

## 常用語法速查

| 你輸入的 | 效果 |
| --- | --- |
| `#工作` | 候選新增「工作」標籤 |
| `@明天` `@下週五` `@3月5日` | 候選設定截止日期 |
| `~下午3點` `~明早9點` | 候選設定提醒時間 |
| `寫報告 @明天 #工作` | 同時候選日期和標籤 |

## 有什麼限制

- `#`、`@`、`~` 最好放在清楚的詞邊界前，例如 `#工作`，不要寫成 `搞#工作`
- 解析結果只是建議，最終以你確認後的欄位為準
- 複雜句子裡夾雜的符號，不一定都能被正確識別

如果某次識別結果不對，直接在欄位裡手動設定就好。自然語言輸入只是快捷方式，不是必須使用的功能。
