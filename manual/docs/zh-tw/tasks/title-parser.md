---

不想每次都在欄位裡一個個點選？試試在標題裡直接寫：

- `#工作` — 新增一個叫「工作」的標籤
- `@明天` — 設定截止日期為明天
- `~下午3點` — 設定提醒時間

GranoFlow 會把這些識別出來，顯示成候選項，等你確認。

## 怎麼操作

輸入標題時，只要寫了 `#` `@` `~` 或日期詞，輸入框會高亮顯示識別結果。

![自然語言解析確認介面](../../../screenshots/zh-tw/tasks-title-parser-confirmation.png)

確認候選項的方式：

- 點擊高亮區域的候選項
- 按 **Enter** 或 **Tab** 確認當前候選

:::caution[重要：確認前不會寫入]
識別到的內容**不會**自動變成欄位。你必須明確確認，才會寫入標籤、日期或提醒。沒有確認的部分會繼續留在標題文字裡，不會偷偷消失。
:::

## 常用語法速查

| 你輸入的 | 效果 |
| --- | --- |
| `#工作` | 候選新增「工作」標籤 |
| `@明天` `@下週五` `@3月5日` | 候選設定截止日期 |
| `~下午3點` `~明早9點` | 候選設定提醒時間 |
| `寫報告 @明天 #工作` | 同時候選日期和標籤 |

## 有什麼限制

- `#` `@` `~` 需要在清楚的詞邊界前使用
- 解析結果只是建議，最終以你確認的欄位為準
- 複雜的句子裡夾雜的符號不一定能被正確識別

如果哪次識別結果不對，直接在欄位裡手動設定就好。
