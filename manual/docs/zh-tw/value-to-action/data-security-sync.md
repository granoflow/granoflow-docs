## 本地優先

本地優先 = 資料先存在你的裝置上，不需要聯網才能用。開啟雲端同步後，才會上傳加密副本到雲端。

## 端對端加密

資料離開裝置前先加密。雲端存的是密文，GranoFlow 的伺服器讀不到你的任務內容。

## 金鑰不是登入密碼

| | 用途 |
|--|------|
| **登入密碼 / 電子郵件驗證碼** | 證明帳號屬於你 |
| **加密金鑰** | 解開你的加密資料 |

你可以順利登入帳號，但沒有正確金鑰，雲端或備份裡的資料仍然解不開。

**金鑰保存建議：** 存進可靠的密碼管理器，換裝置前確認金鑰可用。

:::caution[空裝置不是資料來源]
新裝置第一次同步時，不要把空裝置當成「新起點」——這會覆蓋掉你的舊資料。
:::
